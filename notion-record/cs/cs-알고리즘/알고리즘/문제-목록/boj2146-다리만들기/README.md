# BOJ2146 - 다리만들기

태그: dp, 최적화

- 섬들의 개수를 체크한다
    - 섬들에 각각 번호를 매겨둔다
    - bfs, dfs는 개인 선호

- 섬들 좌표 하나하나를 골라서 4방탐색으로 bfs를 실시한다
    - 범위 안에 들어와있지 않거나, 자기 자신과 같은 섬이거나, 해당 지점에 이미 더 짧은 다리가 있다면 PASS
    - 위의 3가지 조건을 PASS하면
        - 다른 섬을 만난 경우
            - dp[temp.x][temp.y] return후 종료
        - 아직 만나지 않은 경우 d
            - dp[nx][ny] = dp[temp.x][temp.y] + 1;
            - 다음 좌표 큐 삽입

- 최적화
    - dp배열을 만들어둔다
    - bfs를 통해서 간 어떤 지점(nx,ny)이 이전 지점(temp.x, temp.y) + 1보다 작은 값을 가지면 탐색  X
