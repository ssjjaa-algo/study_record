# BOJ16719 - ZOAC

태그: dfs, 이분 탐색

[16719번: ZOAC](https://www.acmicpc.net/problem/16719)

- 난이도 자체는 어렵지 않았으나 구현에 시간이 생각보다 좀 걸림
    - 생각은 느리지 않다고 생각하는데,
    - **`이분 탐색과 재귀의 형식이 겹쳐있는 유형`**에 대한 **`구현`**이 약한 듯 하다.
- 나중에 다시 풀어보는 정도로 넘어간다.

1. 가장 작은 문자를 찾는다

1. 작은 문자를 찾았다면방문 표시
    - 작은 문자를 찾지 못한 경우는 return해야 한다.
        - 하지 않으면 계속 재귀를 돈다. min = Integer.Max_Value인 경우 return

1. 방문표시한 원소들에 대해서 전체 출력

1. 방문한 지점으로부터 오른쪽 / 왼쪽 순으로 분할
