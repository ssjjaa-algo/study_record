# BOJ1561 - 놀이공원

태그: 이분 탐색

[1561번: 놀이 공원](https://www.acmicpc.net/problem/1561)

# 내가 느낀 문제점

## 이분 탐색으로 굳이 **`정확한 값을 안 찾아도 된다.`**

- 범위를 찾는 것만으로 충분한 경우였음.

- 문제의 예제를 노트로 그려보니 규칙이 보였다.
    - **~~이거는 만약 코테에서 노트 쓰지 마라고 하고 냈으면 박살임~~**
- 그래서 이분 탐색으로 풀어야 한다는 것을 catch
    - 여기서 정확한 값을 찾아야 된다는 생각에 사로잡혀서
        - 범위를 찾아내서 알아내려는 것이 아닌 답을 정확히 찾아가는 방법을 고민했다.
            - 선택된 분을 기준으로 다음과 이전을 파악하여 안에 들어와있음을 확인해도
            - 결국 ‘정확’한 값은 얻어내지 못한다.

## 이 문제는 범위를 찾아내어 거기서 하나하나씩 확인하는 것.

- 생각을 확장하자..
