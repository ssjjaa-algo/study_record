# 5. Stack

태그: 큐 1개 스택 2개

1. 개념
2. 특징
3. 활용

개념

- **`후입선출(LIFO)`**을 기본으로 하는 자료구조
    - 나중에 들어온 데이터가, 먼저 나가는 자료구조

특징

- 삽입에 O(1)의 시간, 삭제에 O(1)의 시간을 가진다.
    - Push : 스택의 맨 뒤에 삽입하면 된다.
    - Pop : 스택의 맨 뒤를 삭제하면 된다.

활용

- 후위 표기법 연산, 괄호 유효성 검사, DFS 등의 활용을 한다.

## 스택 2개를 이용해서 큐를 구현해보아라.

- 삽입 enqueStack
    - 들어오는 데이터를 모두 push한다.

- 삭제 dequeStack
    - dequeStack이 비어있는 경우
        - enqueStack의 모든 데이터를 pop하여 집어넣는다.
        - 그 후 dequeStack을 pop하면 처음 들어간 원소가 마지막에 위치하기 때문에 FIFO
            - 비어있는 경우는 시간복잡도  **`O(n)`**
    
    - dequeStack이 비어있지 않은 경우
        - pop한다
            - 비어있지 않은 경우 시간복잡도 **`O(1)`**
