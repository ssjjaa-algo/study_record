# 3. Array VS LinkedList

- Dynamic Array
    - 장점
        - 데이터 접근, 할당이 O(1)
        - 맨 뒤 데이터 추가, 삭제 O(1)
    - 단점
        - 맨 끝이 아닌 곳에 데이터 삽입 삭제 O(n)
        - resize에 시간이 많이 걸림, 사용하지 않고 있는 낭비되는 메모리 공간 발생

- LinkedList
    - 장점
        - O(1)으로 삽입/삭제
        - 데이터 들어오는 양 예측 불가시
        - 조회 작업 별로 하지 않을 때
