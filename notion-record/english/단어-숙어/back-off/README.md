# back off

`back off`는 회화에서 자주 쓰는 구동사이고, 핵심 뜻은 **물러서다 / 그만하다 / 간섭하지 않다**이다.

```
back off
물러서다 / 그만하다 / 간섭하지 마
```

# 1. 물리적으로 물러서다

누군가에게 너무 가까이 다가왔을 때 **뒤로 물러서다**라는 뜻으로 쓴다.

```
Back off.
물러서.
```

```
Back off a little.
조금 물러서.
```

```
The dog growled, so I backed off.
개가 으르렁거려서 나는 뒤로 물러섰다.
```

# 2. 간섭하지 마 / 참견하지 마

상대가 너무 밀어붙이거나 간섭할 때 **그만해 / 참견하지 마 / 내버려 둬**라는 뜻으로 쓴다.

```
Back off. This is none of your business.
그만해. 이건 네가 상관할 일이 아니야.
```

```
I told him to back off.
나는 그에게 그만 간섭하라고 말했다.
```

# 3. 압박을 줄이다 / 강도를 낮추다

누군가에게 압박을 주다가 **조금 누그러뜨리다 / 압박을 줄이다**라는 뜻으로도 쓴다.

```
You need to back off and give her some space.
너는 좀 물러서서 그녀에게 공간을 줘야 해.
```

```
The manager finally backed off.
매니저가 마침내 압박을 좀 줄였다.
```

# 4. 명령문으로 쓸 때의 느낌

`Back off`는 짧고 강한 명령이다.

```
Back off!
꺼져 / 그만해 / 물러서!
```

상황에 따라 꽤 공격적으로 들릴 수 있다. 부드럽게 말하고 싶으면 다음처럼 말할 수 있다.

```
Could you give me some space?
공간 좀 줄 수 있어?
```

```
Please stop pushing me.
그만 몰아붙여줘.
```

# 5. 비슷한 표현

| 표현 | 뜻 | 느낌 |
| --- | --- | --- |
| back off | 물러서다 / 그만하다 | 직접적, 강함 |
| step back | 한 걸음 물러서다 | 물리적/비유적 모두 가능 |
| leave me alone | 나 좀 내버려 둬 | 개인적 감정이 강함 |
| give me some space | 공간 좀 줘 | 비교적 부드러움 |
| stop pushing me | 그만 몰아붙여 | 압박을 멈추라는 느낌 |

# 6. 예문 모음

```
Back off, Dwight.
물러서, 드와이트.
```

```
I said back off.
물러서라고 했잖아.
```

```
She told him to back off.
그녀는 그에게 그만하라고 말했다.
```

```
You should back off for now.
지금은 좀 물러서는 게 좋겠어.
```

```
He backed off when he realized I was serious.
그는 내가 진심이라는 걸 깨닫고 물러섰다.
```

# 7. 핵심 정리

| 표현 | 뜻 | 예문 |
| --- | --- | --- |
| back off | 물러서다 | Back off a little. |
| back off | 그만하다 / 간섭하지 않다 | Tell him to back off. |
| back off | 압박을 줄이다 | You need to back off. |

가장 중요한 감각은 이것이다.

```
back off = 가까이 오거나 압박하던 것을 멈추고 뒤로 빠지다
```

따라서 단순히 몸을 뒤로 빼는 뜻도 되고, 비유적으로 **간섭하지 마 / 그만 몰아붙여**라는 뜻도 된다.
