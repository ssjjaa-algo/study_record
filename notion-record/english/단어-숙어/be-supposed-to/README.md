# be supposed to

# be supposed to 정리

`be supposed to`는 회화에서 정말 자주 쓰이는 표현이고, 핵심 뜻은 **원래 ~하기로 되어 있다 / ~해야 한다 / ~라고 여겨진다**이다.

```
be supposed to + 동사원형
~하기로 되어 있다 / ~해야 한다 / ~라고 되어 있다
```

# 1. 예정되어 있다 / 하기로 되어 있다

가장 기본적인 뜻은 **원래 계획이나 약속상 ~하기로 되어 있다**이다.

```
I’m supposed to meet him at 3.
나는 3시에 그를 만나기로 되어 있어.
```

```
The meeting is supposed to start at 10.
회의는 10시에 시작하기로 되어 있어.
```

```
We’re supposed to be there by noon.
우리는 정오까지 거기에 도착하기로 되어 있어.
```

# 2. ~해야 한다 / 의무가 있다

규칙, 지시, 사회적 기대 때문에 **~해야 한다**는 뜻으로도 쓰인다.

```
You’re supposed to wear a seat belt.
너는 안전벨트를 매야 해.
```

```
Employees are supposed to clock in before 9.
직원들은 9시 전에 출근 기록을 해야 한다.
```

```
I’m supposed to finish this report today.
나는 오늘 이 보고서를 끝내야 해.
```

`must`보다 약간 덜 직접적이고, **규칙상/기대상 그렇게 해야 한다**는 느낌이다.

# 3. 원래는 ~해야 하는데 실제로는 아닐 수 있음

`be supposed to`는 자주 **원래는 그래야 하는데, 실제로는 아닐 수도 있다**는 뉘앙스를 가진다.

```
This printer is supposed to work.
이 프린터는 원래 작동해야 하는데.
```

실제로는 작동하지 않는 상황일 수 있다.

```
He’s supposed to be here.
그는 원래 여기 있어야 하는데.
```

그런데 아직 안 왔거나, 안 보이는 상황일 수 있다.

```
You were supposed to call me.
너 나한테 전화하기로 했잖아.
```

실제로 전화하지 않았다는 불만이 담길 수 있다.

# 4. ~라고 한다 / ~라고 알려져 있다

정보나 소문, 일반적 평가를 말할 때 **~라고 한다 / ~라고 여겨진다**는 뜻도 된다.

```
This restaurant is supposed to be good.
이 식당 좋다고 하더라.
```

```
The movie is supposed to be funny.
그 영화 웃기다고 하더라.
```

```
He’s supposed to be an expert.
그는 전문가라고 알려져 있어.
```

이때는 말하는 사람이 직접 확신한다기보다, **들은 정보나 평판**을 전달하는 느낌이다.

# 5. 부정형: be not supposed to

부정형은 **~하면 안 된다 / ~하기로 되어 있지 않다**라는 뜻이다.

```
You’re not supposed to smoke here.
여기서 담배 피우면 안 돼.
```

```
We’re not supposed to tell anyone.
우리는 아무에게도 말하면 안 돼.
```

```
I’m not supposed to eat sugar.
나는 설탕을 먹으면 안 돼.
```

# 6. 과거형: was/were supposed to

과거형은 **~하기로 되어 있었다 / ~했어야 했다**라는 뜻이다.

```
I was supposed to meet him yesterday.
나는 어제 그를 만나기로 되어 있었다.
```

```
You were supposed to be here at 8.
너는 8시에 여기 있기로 했잖아.
```

```
She was supposed to send the email.
그녀가 그 이메일을 보내기로 되어 있었다.
```

문맥에 따라 실제로 했는지 안 했는지는 달라진다. 하지만 자주 **안 했다는 불만**을 담는다.

# 7. 의문문

```
Am I supposed to wait here?
제가 여기서 기다려야 하나요?
```

```
What am I supposed to do?
내가 뭘 해야 하는데?
```

```
Are we supposed to bring anything?
우리가 뭘 가져가야 하나요?
```

`What am I supposed to do?`는 상황에 따라 **나보고 어쩌라는 거야?**라는 불만 섞인 표현도 된다.

# 8. have to / should / be supposed to 차이

| 표현 | 느낌 | 예문 |
| --- | --- | --- |
| have to | 현실적 필요, 강한 의무 | I have to go. |
| should | 조언, 권유, 약한 의무 | You should rest. |
| be supposed to | 규칙, 예정, 기대 | You’re supposed to be here. |

## have to

```
I have to leave now.
나 지금 떠나야 해.
```

실제로 해야 하는 상황, 필요성이 강하다.

## should

```
You should call her.
그녀에게 전화하는 게 좋겠어.
```

조언이나 권유 느낌이다.

## be supposed to

```
You’re supposed to call her.
너 그녀에게 전화하기로 되어 있잖아.
```

이미 정해진 약속, 규칙, 기대가 있다는 느낌이다.

# 9. 자주 쓰는 패턴

| 표현 | 뜻 |
| --- | --- |
| I’m supposed to ~ | 나는 ~하기로 되어 있다 / ~해야 한다 |
| You’re supposed to ~ | 너는 ~해야 한다 / ~하기로 되어 있다 |
| We’re supposed to ~ | 우리는 ~하기로 되어 있다 |
| It’s supposed to ~ | 그것은 ~하기로 되어 있다 / ~라고 한다 |
| This is supposed to ~ | 이건 원래 ~해야 한다 |
| What am I supposed to ~? | 내가 뭘 ~해야 하는데? |
| not supposed to ~ | ~하면 안 된다 |

# 10. 예문 모음

```
I’m supposed to work tomorrow.
나는 내일 일하기로 되어 있어.
```

```
You’re supposed to knock before entering.
들어가기 전에 노크해야 해.
```

```
This medicine is supposed to help you sleep.
이 약은 네가 잠드는 데 도움이 된다고 되어 있어.
```

```
The bus is supposed to arrive in five minutes.
버스는 5분 후에 도착할 예정이야.
```

```
You’re not supposed to touch that.
그거 만지면 안 돼.
```

```
What are we supposed to do now?
이제 우리가 뭘 해야 하는데?
```

# 11. 핵심 정리

| 뜻 | 예문 | 해석 |
| --- | --- | --- |
| ~하기로 되어 있다 | I’m supposed to meet him. | 나는 그를 만나기로 되어 있다 |
| ~해야 한다 | You’re supposed to wear this. | 너는 이걸 착용해야 한다 |
| 원래는 ~해야 한다 | This is supposed to work. | 이건 원래 작동해야 한다 |
| ~라고 한다 | It’s supposed to be good. | 그거 좋다고 한다 |
| ~하면 안 된다 | You’re not supposed to smoke here. | 여기서 담배 피우면 안 된다 |

가장 중요한 감각은 이것이다.

```
be supposed to = 원래 정해진 기대/규칙/예정이 있다
```

그래서 `I’m supposed to go`는 단순히 “가야 해”가 아니라, **원래 가기로 되어 있다 / 가야 하는 상황이다**라는 느낌이다.
