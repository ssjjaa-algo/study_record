# upside down 정리

`upside down`은 단어처럼 외워도 되고 숙어처럼 외워도 되는 표현이다. 핵심 뜻은 **위아래가 뒤집힌 / 거꾸로**이다.

```
upside down
거꾸로 / 위아래가 뒤집혀서
```

# 1. 기본 뜻: 거꾸로, 위아래가 뒤집힌

물건이 정상 방향이 아니라 위아래가 뒤집혀 있을 때 쓴다.

```
The picture is upside down.
그 사진이 거꾸로 되어 있어.
```

```
You’re holding the book upside down.
너 책을 거꾸로 들고 있어.
```

```
The map is upside down.
지도가 거꾸로 되어 있어.
```

# 2. turn something upside down

`turn something upside down`은 **~을 거꾸로 뒤집다**라는 뜻이다.

```
He turned the box upside down.
그는 상자를 거꾸로 뒤집었다.
```

```
Don’t turn the bottle upside down.
그 병 거꾸로 뒤집지 마.
```

# 3. 비유적 뜻: 완전히 뒤흔들다 / 엉망으로 만들다

`upside down`은 비유적으로 **정상적인 상태를 완전히 뒤집어 놓다 / 혼란스럽게 만들다**라는 뜻도 있다.

```
The news turned my world upside down.
그 소식은 내 세상을 완전히 뒤흔들었다.
```

```
The kids turned the room upside down.
아이들이 방을 완전히 엉망으로 만들어 놓았다.
```

# 4. search / look through와 함께 쓰일 때

무언가를 찾으려고 집이나 방을 다 뒤질 때도 쓴다.

```
I turned the house upside down looking for my keys.
열쇠를 찾느라 집을 샅샅이 뒤졌다.
```

# 5. upside down vs inside out

| 표현 | 뜻 | 차이 |
| --- | --- | --- |
| upside down | 위아래가 뒤집힌 | 방향이 거꾸로 됨 |
| inside out | 안팎이 뒤집힌 | 안쪽과 바깥쪽이 뒤집힘 |

```
Your shirt is inside out.
네 셔츠 안팎이 뒤집혔어.
```

```
Your book is upside down.
네 책이 거꾸로 되어 있어.
```

# 6. 형용사로 명사 앞에 쓸 때

명사 앞에서 형용사처럼 쓰면 보통 하이픈을 붙여 `upside-down`으로 쓴다.

```
an upside-down picture
거꾸로 된 사진
```

```
an upside-down cake
뒤집은 케이크
```

문장 뒤에서 보어처럼 쓰면 보통 하이픈 없이 `upside down`으로 쓴다.

```
The picture is upside down.
그 사진은 거꾸로 되어 있다.
```

# 7. 예문 모음

```
It’s upside down.
그거 거꾸로야.
```

```
You put it upside down.
너 그거 거꾸로 놨어.
```

```
My life was turned upside down.
내 삶이 완전히 뒤흔들렸다.
```

```
They turned the office upside down looking for the file.
그들은 그 파일을 찾느라 사무실을 샅샅이 뒤졌다.
```

# 8. 핵심 정리

| 표현 | 뜻 | 예문 |
| --- | --- | --- |
| upside down | 거꾸로 / 위아래가 뒤집힌 | The picture is upside down. |
| turn A upside down | A를 거꾸로 뒤집다 | Turn the box upside down. |
| turn my world upside down | 내 세상을 완전히 뒤흔들다 | The news turned my world upside down. |
| turn the room upside down | 방을 엉망으로 만들다 / 샅샅이 뒤지다 | The kids turned the room upside down. |

가장 중요한 감각은 이것이다.

```
upside down = 정상 방향이나 상태가 완전히 뒤집힌 것
```
