# Affirmative

`Affirmative`는 기본적으로 **긍정의 / 그렇다 / 맞다 / 알겠다**라는 뜻이다.

```
affirmative
긍정의 / 그렇다 / 맞다
```

# 1. 대답으로 쓰일 때

`Affirmative`는 특히 무전, 군대식 말투, 공식적인 통신에서 **Yes**의 의미로 쓰인다.

```
Affirmative.
그렇다 / 맞다 / 알겠다.
```

일상 대화에서 매일 쓰는 표현이라기보다는, 약간 딱딱하고 군대식, 로봇식, 무전식 느낌이 있다.

# 2. Yes와의 차이

| 표현 | 뜻 | 느낌 |
| --- | --- | --- |
| Yes | 네 / 맞아 | 일반적 |
| Yeah | 응 / 맞아 | 캐주얼 |
| Affirmative | 그렇다 / 확인했다 | 공식적, 군대식, 무전식 |

예문:

```
Are you in position?
Affirmative.
```

해석:

```
위치에 있나?
확인. / 그렇다.
```

# 3. 형용사로 쓰일 때

`affirmative`는 형용사로 **긍정적인 / 긍정의**라는 뜻도 있다.

```
He gave an affirmative answer.
그는 긍정적인 대답을 했다.
```

```
She nodded in the affirmative.
그녀는 그렇다는 뜻으로 고개를 끄덕였다.
```

# 4. 부정형과 반대 표현

`affirmative`의 반대는 `negative`다.

| 표현 | 뜻 |
| --- | --- |
| affirmative | 그렇다 / 긍정 |
| negative | 아니다 / 부정 |

무전식 표현에서는 다음처럼 쓸 수 있다.

```
Affirmative.
그렇다.
```

```
Negative.
아니다.
```

# 5. 예문 모음

```
Affirmative, sir.
알겠습니다, sir.
```

```
Mission confirmed? Affirmative.
임무 확인됐나? 그렇다.
```

```
The answer is affirmative.
답은 긍정이다.
```

```
He responded in the affirmative.
그는 그렇다고 대답했다.
```

# 6. 핵심 정리

| 표현 | 뜻 | 설명 |
| --- | --- | --- |
| Affirmative. | 그렇다 / 맞다 / 알겠다 | Yes의 공식적/무전식 표현 |
| affirmative answer | 긍정적인 대답 | 형용사 사용 |
| in the affirmative | 긍정으로 / 그렇다고 | 격식 있는 표현 |
| Negative. | 아니다 | affirmative의 반대 |

가장 중요한 감각은 이것이다.

```
Affirmative = Yes를 공식적이고 딱딱하게 말하는 느낌
```

그래서 일상 대화에서는 보통 `yes`, `yeah`, `right`를 더 많이 쓰고, `Affirmative`는 군대식/무전식/장난스럽게 딱딱한 말투를 낼 때 자주 보인다.
