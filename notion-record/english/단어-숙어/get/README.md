# get

# get의 여러 뜻 정리

`get`은 영어에서 가장 많이 쓰이는 동사 중 하나이고, 기본 이미지는 **어떤 상태나 대상에 도달하다 / 손에 넣다 / 변화를 겪다**이다.

```
get = 얻다 / 도착하다 / 되다 / 이해하다 / 시키다
```

# 1. 얻다 / 받다

가장 기본적인 뜻은 **무언가를 얻다, 받다, 구하다**이다.

```
I got a message from her.
나는 그녀에게서 메시지를 받았다.
```

```
Can I get some water?
물 좀 받을 수 있을까요?
```

# 2. 사다 / 구입하다

일상 회화에서는 `buy` 대신 `get`을 써서 **사다, 구하다**라는 뜻으로 자주 말한다.

```
I got a new phone.
나 새 휴대폰 샀어.
```

```
We need to get a cake.
우리 케이크 하나 사야 해.
```

# 3. 도착하다

`get to + 장소`는 **~에 도착하다**라는 뜻이다.

```
What time did you get home?
너 몇 시에 집에 도착했어?
```

```
We got to the office late.
우리는 사무실에 늦게 도착했다.
```

주의할 점은 `home`은 부사처럼 쓰이기 때문에 보통 `to`를 쓰지 않는다.

```
get home
집에 도착하다
```

# 4. ~한 상태가 되다

`get + 형용사`는 **~해지다 / ~한 상태가 되다**라는 뜻이다.

```
It’s getting dark.
어두워지고 있어.
```

```
I got tired.
나는 피곤해졌다.
```

```
Don’t get angry.
화내지 마.
```

# 5. 이해하다

회화에서 `get it`은 **이해하다**라는 뜻이다.

```
I get it.
이해했어.
```

```
I don’t get what you mean.
네가 무슨 뜻인지 모르겠어.
```

```
Do you get the joke?
그 농담 이해했어?
```

# 6. 가져오다 / 데려오다

`get someone/something`은 문맥에 따라 **가져오다, 데려오다**라는 뜻이 된다.

```
Can you get my bag?
내 가방 좀 가져와 줄래?
```

```
I’ll get the kids from school.
내가 학교에서 아이들을 데려올게.
```

# 7. ~하게 만들다

`get + O + 형용사/과거분사`는 **O를 어떤 상태가 되게 만들다**라는 뜻이다.

```
This will get you ready.
이게 너를 준비되게 해줄 거야.
```

```
That could get me arrested.
그건 나를 체포되게 만들 수도 있어.
```

```
We need to get this done.
우리는 이걸 끝내야 해.
```

# 8. ~하게 시키다 / 설득하다

`get + 사람 + to부정사`는 **누군가가 ~하게 만들다 / 설득하다 / 시키다**라는 뜻이다.

```
I got him to help me.
나는 그가 나를 돕게 했다.
```

```
Can you get her to call me?
그녀가 나에게 전화하게 해줄 수 있어?
```

`make / have / let` 같은 사역동사는 보통 동사원형을 쓰지만, `get`은 `to부정사`를 쓴다.

```
make him go
have him go
let him go
get him to go
```

# 9. 연락하다 / 닿다

`get someone`은 전화나 연락에서 **연락이 닿다, 연결되다**라는 뜻으로도 쓰인다.

```
I couldn’t get him on the phone.
전화로 그에게 연락이 안 됐다.
```

```
Can you get me later?
나중에 나한테 연락해줄래?
```

# 10. 잡다 / 붙잡다 / 맞히다

문맥에 따라 `get`은 **잡다, 붙잡다, 맞히다, 당하게 하다**라는 뜻이 된다.

```
The police got him.
경찰이 그를 잡았다.
```

```
The bullet got him in the leg.
총알이 그의 다리에 맞았다.
```

```
You got me.
네가 날 잡았네 / 들켰네 / 모르겠어.
```

# 11. 먹다 / 주문하다

식당이나 카페에서 `get`은 **먹다, 주문하다**라는 뜻으로 자주 쓰인다.

```
I’ll get a coffee.
저는 커피 하나 할게요.
```

```
What are you getting?
너 뭐 먹을 거야? / 뭐 주문할 거야?
```

# 12. 병에 걸리다

`get + 병명`은 **병에 걸리다**라는 뜻이다.

```
I got a cold.
나는 감기에 걸렸다.
```

```
She got the flu.
그녀는 독감에 걸렸다.
```

# 13. 수동의 의미: get + p.p.

`get + 과거분사`는 `be + p.p.`처럼 수동 의미를 만들 수 있다. 보통 사건이나 변화가 일어났다는 느낌이 강하다.

```
He got fired.
그는 해고당했다.
```

```
They got married.
그들은 결혼했다.
```

```
My phone got stolen.
내 휴대폰이 도난당했다.
```

# 14. 기회를 얻다 / ~하게 되다

`get to + 동사원형`은 **~할 기회를 얻다 / ~하게 되다**라는 뜻이다.

```
I got to meet him.
나는 그를 만날 기회를 얻었다.
```

```
Did you get to see the movie?
그 영화 볼 수 있었어?
```

# 15. 핵심 구동사 표현

| 표현 | 뜻 | 예문 |
| --- | --- | --- |
| get up | 일어나다 | I get up at 7. |
| get back | 돌아오다 | I’ll get back soon. |
| get back at | 복수하다 | I want to get back at him. |
| get back to | 다시 연락하다 | I’ll get back to you later. |
| get over | 극복하다 | She got over the breakup. |
| get along with | ~와 잘 지내다 | I get along with my coworkers. |
| get away | 벗어나다, 도망가다 | He got away. |
| get away with | 벌받지 않고 넘어가다 | He got away with lying. |
| get into | ~에 들어가다, 빠지다 | I got into programming. |
| get out of | ~에서 나오다, 피하다 | He got out of the car. |

# 16. 핵심 정리

| get의 뜻 | 예문 | 해석 |
| --- | --- | --- |
| 얻다/받다 | I got a message. | 메시지를 받았다 |
| 사다 | I got a phone. | 휴대폰을 샀다 |
| 도착하다 | I got home. | 집에 도착했다 |
| ~해지다 | It got dark. | 어두워졌다 |
| 이해하다 | I get it. | 이해했다 |
| 가져오다 | Get my bag. | 내 가방 가져와 |
| ~하게 만들다 | It got me angry. | 그게 나를 화나게 했다 |
| ~하게 시키다 | I got him to help. | 그가 돕게 했다 |
| 병에 걸리다 | I got a cold. | 감기에 걸렸다 |
| 수동 | He got fired. | 그는 해고당했다 |

가장 중요한 감각은 이것이다.

```
get = 어떤 상태나 대상에 도달하다
```

그래서 `get a cake`는 케이크를 손에 넣는 것이고, `get tired`는 피곤한 상태에 도달하는 것이고, `get home`은 집이라는 장소에 도달하는 것이다.
