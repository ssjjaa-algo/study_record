# over here

`over here`는 회화에서 자주 쓰는 위치 표현이고, 핵심 뜻은 **여기 이쪽으로 / 여기 이쪽에**이다.

```
over here
여기 이쪽에 / 여기로
```

# 1. 여기 이쪽에

말하는 사람이 있는 쪽이나, 손짓으로 가리키는 가까운 쪽을 나타낸다.

```
I’m over here.
나 여기 있어.
```

```
The keys are over here.
열쇠는 여기 이쪽에 있어.
```

```
Come over here.
이리 와.
```

# 2. here와 over here의 차이

| 표현 | 뜻 | 느낌 |
| --- | --- | --- |
| here | 여기 | 단순히 현재 위치 |
| over here | 여기 이쪽 | 상대의 시선을 이쪽으로 끌거나, 약간 떨어진 이쪽 위치를 강조 |

예를 들어 멀리 있는 사람에게 부를 때는 `here`보다 `over here`가 더 자연스러울 수 있다.

```
Hey, over here!
야, 여기야!
```

# 3. over의 느낌

`over`는 여기서 반드시 “위에”라는 뜻은 아니다. 공간상으로 **저쪽에서 이쪽으로 이동하거나 시선을 옮기는 느낌**을 준다.

```
Come over here.
이쪽으로 와.
```

즉, 상대방의 현재 위치에서 말하는 사람 쪽으로 오라는 느낌이다.

# 4. 자주 쓰는 표현

| 표현 | 뜻 |
| --- | --- |
| over here | 여기 이쪽에 |
| come over here | 이리 와 |
| right over here | 바로 여기 이쪽에 |
| over there | 저쪽에 |
| come over | 들르다 / 오다 |

# 5. over here vs over there

| 표현 | 의미 |
| --- | --- |
| over here | 말하는 사람 쪽, 여기 이쪽 |
| over there | 말하는 사람에게서 떨어진 저쪽 |

```
Come over here.
이리 와.
```

```
Go over there.
저쪽으로 가.
```

# 6. 예문 모음

```
Over here!
여기야!
```

```
Can you come over here for a second?
잠깐 이쪽으로 와줄래?
```

```
Put the box over here.
상자를 여기 이쪽에 둬.
```

```
We’re sitting over here.
우리는 여기 이쪽에 앉아 있어.
```

```
The problem is over here.
문제는 여기 이쪽에 있어.
```

# 7. 핵심 정리

| 표현 | 뜻 | 예문 |
| --- | --- | --- |
| here | 여기 | I’m here. |
| over here | 여기 이쪽 | I’m over here. |
| come over here | 이리 와 | Come over here. |
| over there | 저쪽에 | It’s over there. |

가장 중요한 감각은 이것이다.

```
over here = 상대의 시선을 말하는 사람 쪽/이쪽 위치로 끌어오는 표현
```

그래서 단순히 “여기”보다 **여기 이쪽이야 / 이리 와 / 여기 봐**라는 느낌이 더 강하다.
