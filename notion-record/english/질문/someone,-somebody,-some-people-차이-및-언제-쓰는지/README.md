# someone, somebody, some people 차이 및 언제 쓰는지

# 질문

someone, somebody, some people 각각 차이 및 언제 쓰는지

# 1. 핵심 차이

| 표현 | 수 | 뜻 | 핵심 느낌 |
| --- | --- | --- | --- |
| --- | ---: | --- | --- |
| someone | 단수 | 누군가 | 특정하지 않은 한 사람 |
| somebody | 단수 | 누군가 | someone과 거의 같고 조금 더 구어적 |
| some people | 복수 | 몇몇 사람들 / 어떤 사람들 | 한 명이 아니라 여러 사람 |

# 2. someone

`someone`은 **누군가 한 사람**을 뜻한다.

```
Someone is at the door.
누군가 문 앞에 있어.
```

```
I need someone to help me.
나를 도와줄 누군가가 필요해.
```

여기서 중요한 점은 **한 사람**이라는 것이다. 누군지 정확히 모르거나 말하지 않을 때 쓴다.

# 3. somebody

`somebody`도 **누군가 한 사람**이라는 뜻이다. 의미는 `someone`과 거의 같다.

```
Somebody called you.
누군가 너에게 전화했어.
```

```
There’s somebody in the room.
방 안에 누군가 있어.
```

`somebody`는 `someone`보다 약간 더 일상적이고 구어적인 느낌이 날 수 있다. 하지만 실제 회화에서는 둘을 거의 바꿔 써도 된다.

# 4. someone vs somebody

| 표현 | 차이 |
| --- | --- |
| someone | 조금 더 중립적, 글에서도 자연스러움 |
| somebody | 조금 더 구어적, 말하는 느낌 |

예문 비교:

```
Someone left their bag here.
누군가 가방을 여기에 두고 갔어.
```

```
Somebody left their bag here.
누군가 가방을 여기에 두고 갔어.
```

둘 다 자연스럽고 의미 차이는 거의 없다.

# 5. some people

`some people`은 **몇몇 사람들 / 어떤 사람들**이라는 뜻이다. 반드시 복수다.

```
Some people like spicy food.
어떤 사람들은 매운 음식을 좋아해.
```

```
Some people don’t understand this rule.
몇몇 사람들은 이 규칙을 이해하지 못해.
```

`someone/somebody`가 한 사람을 가리킨다면, `some people`은 여러 사람 또는 사람들의 일부 집단을 가리킨다.

# 6. 비교 예문

## 6-1. 한 사람인지 여러 사람인지

```
Someone is waiting outside.
누군가 한 사람이 밖에서 기다리고 있어.
```

```
Somebody is waiting outside.
누군가 한 사람이 밖에서 기다리고 있어.
```

```
Some people are waiting outside.
몇몇 사람들이 밖에서 기다리고 있어.
```

차이는 수다.

```
someone / somebody = 한 사람
some people = 여러 사람
```

# 7. 동사 형태 차이

`someone`과 `somebody`는 단수 취급한다.

```
Someone is here.
누군가 여기 있어.
```

```
Somebody wants to talk to you.
누군가 너와 얘기하고 싶어 해.
```

`some people`은 복수 취급한다.

```
Some people are here.
몇몇 사람들이 여기 있어.
```

```
Some people want to talk to you.
몇몇 사람들이 너와 얘기하고 싶어 해.
```

# 8. 언제 어떤 것을 쓰나

## 8-1. 누군지 모르는 한 사람

```
Someone called me.
누군가 나한테 전화했어.
```

```
Somebody took my pen.
누군가 내 펜을 가져갔어.
```

이때는 `someone` 또는 `somebody`를 쓴다.

## 8-2. 특정하지 않은 여러 사람

```
Some people think this is easy.
어떤 사람들은 이게 쉽다고 생각해.
```

```
Some people don’t like coffee.
어떤 사람들은 커피를 좋아하지 않아.
```

이때는 `some people`을 쓴다.

## 8-3. 사람 일반 중 일부를 말할 때

```
Some people are very sensitive about grammar.
어떤 사람들은 문법에 매우 민감해.
```

이건 특정 한 사람이 아니라, 사람들 중 일부 그룹을 말하는 것이다.

# 9. 한국어로 헷갈리는 이유

한국어의 **누군가 / 어떤 사람들 / 몇몇 사람들**이 문맥에 따라 비슷하게 느껴질 수 있다.

하지만 영어에서는 수가 중요하다.

| 한국어 느낌 | 영어 |
| --- | --- |
| 누군가 한 명 | someone / somebody |
| 어떤 사람들, 몇몇 사람들 | some people |

# 10. 핵심 정리

```
someone = 누군가 한 사람
somebody = 누군가 한 사람, someone보다 조금 더 구어적
some people = 몇몇 사람들, 여러 사람
```

따라서:

```
Someone is coming.
누군가 한 사람이 오고 있다.
```

```
Some people are coming.
몇몇 사람들이 오고 있다.
```

이 둘은 의미가 다르다.

# 11. 추가 팁

`someone`과 `somebody`는 뒤에서 대명사로 받을 때 현대 영어에서는 성별을 모르면 보통 `they/their`를 쓴다.

```
Someone left their phone here.
누군가 자기 휴대폰을 여기에 두고 갔어.
```

문법적으로 `someone`은 단수지만, 성별을 특정하지 않을 때 `their`를 쓰는 것이 자연스럽다.
