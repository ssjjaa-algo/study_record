# We can’t write our diseases down.

# 문장

We can’t write our diseases down.

# 질문

`write down our diseases`로 쓰면 안 되는가?  

`write our diseases down`과 차이가 무엇인가?

# 1. 전체 의미

**“우리는 우리의 병명을 적어둘 수 없어.”**

또는 문맥에 따라:

**“우리는 우리가 가진 병들을 써 내려갈 수 없어.”**

여기서 `write down`은 **적다, 기록하다**라는 뜻의 구동사다.

# 2. 결론

둘 다 가능하다.

```
We can’t write our diseases down.
```

```
We can’t write down our diseases.
```

둘 다 문법적으로 맞고, 의미도 거의 같다.

다만 차이는 **목적어의 위치**다.

# 3. write down의 구조

`write down`은 구동사다.

```
write down = 적다 / 기록하다
```

구동사 중에는 목적어가 중간에 들어갈 수도 있고, 뒤에 올 수도 있는 것이 있다.

```
write + 목적어 + down
```

또는

```
write down + 목적어
```

그래서 다음 두 문장은 모두 가능하다.

```
write our diseases down
```

```
write down our diseases
```

# 4. 문장 성분 표시

## 4-1. We can’t write our diseases down.

```
We can’t write our diseases down.
S      V          O           particle
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | We | 주어 |
| V | can’t write down | 기록할 수 없다 |
| O | our diseases | 우리의 질병들 |
| particle | down | 구동사 `write down`의 일부 |

이 문장은 크게 보면:

```
S + V + O
We + can’t write down + our diseases
```

즉 **3형식**으로 볼 수 있다.

## 4-2. We can’t write down our diseases.

```
We can’t write down our diseases.
S      V              O
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | We | 주어 |
| V | can’t write down | 기록할 수 없다 |
| O | our diseases | 우리의 질병들 |

이것도 마찬가지로 **3형식**이다.

```
S + V + O
```

# 5. 두 표현의 차이

| 표현 | 구조 | 느낌 |
| --- | --- | --- |
| write our diseases down | write + O + down | 목적어 `our diseases`가 조금 더 앞에 드러남 |
| write down our diseases | write down + O | `write down`이라는 동작이 먼저 묶여 보임 |

의미 차이는 거의 없다.

둘 다:

```
우리의 질병들을 적다 / 기록하다
```

라는 뜻이다.

# 6. 중요한 규칙: 목적어가 대명사일 때

목적어가 명사일 때는 둘 다 가능하다.

```
write our diseases down
write down our diseases
```

하지만 목적어가 대명사일 때는 반드시 중간에 넣어야 한다.

```
write them down
```

이건 가능하다.

하지만:

```
write down them
```

이건 부자연스럽고 틀린 표현이다.

즉:

```
명사 목적어 → 중간 / 뒤 둘 다 가능
대명사 목적어 → 반드시 중간
```

# 7. 예문 비교

## 명사 목적어

```
Write your name down.
네 이름을 적어.
```

```
Write down your name.
네 이름을 적어.
```

둘 다 가능하다.

## 대명사 목적어

```
Write it down.
그거 적어.
```

가능하다.

```
Write down it.
```

불가능하거나 매우 부자연스럽다.

# 8. disease / diseases 주의

`disease`는 **질병**이라는 뜻이다.

```
disease
질병
```

```
diseases
질병들
```

질문에서 쓴 `diesease`는 철자가 틀린 형태이고, 올바른 철자는 `disease`다.

# 9. 핵심 정리

```
We can’t write our diseases down.
```

가능하다.

```
We can’t write down our diseases.
```

이것도 가능하다.

둘 다 **S + V + O**, 즉 **3형식**이다.

차이는 거의 없고, `our diseases`처럼 명사 목적어일 때는 위치를 바꿀 수 있다.

하지만 `them` 같은 대명사로 바꾸면 반드시:

```
We can’t write them down.
```

이라고 해야 한다.

```
We can’t write down them.
```

은 자연스럽지 않다.
