# Of course, by saying that, she was gambling that I wouldn’t smack her

# 문장

Of course, by saying that, she was gambling that I wouldn’t smack her.

# 1. 전체 의미

**“물론, 그렇게 말함으로써 그녀는 내가 자신을 때리지 않을 거라고 도박을 건 셈이었어.”**

더 자연스럽게는:

**“물론, 그렇게 말한 건 내가 자기를 때리진 않을 거라고 믿고 위험을 감수한 거였지.”**

# 2. 뉘앙스

`Of course`는 **물론 / 당연히**라는 뜻으로, 말하는 사람이 상황을 이미 알고 있다는 느낌을 준다.

`by saying that`은 **그렇게 말함으로써**라는 뜻이다. 어떤 행동의 방식이나 수단을 나타낸다.

`she was gambling that ~`은 문자 그대로는 **~라고 도박하고 있었다**이지만, 실제로는 **~라고 믿고 위험을 감수하고 있었다**라는 뜻이다.

`smack her`는 **그녀를 때리다 / 찰싹 때리다**라는 뜻이다. 문맥에 따라 실제 폭력이라기보다 과장되거나 농담 섞인 표현일 수도 있다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| Of course | 물론 / 당연히 |
| by saying that | 그렇게 말함으로써 |
| she was gambling | 그녀는 위험을 감수하고 있었다 / 도박을 걸고 있었다 |
| that I wouldn’t smack her | 내가 그녀를 때리지 않을 거라고 |

# 4. 문장 성분 표시

```
Of course, by saying that, she was gambling that I wouldn’t smack her.
                              S    V            O(that절)
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| 부사 | Of course | 문장 전체에 대한 말하는 사람의 태도 |
| 부사구 | by saying that | 그렇게 말함으로써, 수단/방식 |
| S | she | 주어 |
| V | was gambling | 동사구, 위험을 감수하고 있었다 |
| O | that I wouldn’t smack her | 목적어 역할의 that절 |

that절 내부 구조:

```
I wouldn’t smack her
S   V             O
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | I | 주어 |
| V | wouldn’t smack | 때리지 않을 것이다 |
| O | her | 목적어, 그녀를 |

# 5. 문장 형식

전체 문장은 크게 보면 **3형식**이다.

```
S + V + O
she + was gambling + that I wouldn’t smack her
```

여기서 목적어는 `that I wouldn’t smack her`라는 명사절이다.

that절 안쪽도 **3형식**이다.

```
S + V + O
I + wouldn’t smack + her
```

# 6. 핵심 문법 포인트

## 6-1. by + V-ing

`by + 동명사`는 **~함으로써**라는 뜻이다.

```
by saying that
그렇게 말함으로써
```

예문:

```
You can improve by practicing every day.
매일 연습함으로써 실력을 향상시킬 수 있다.
```

## 6-2. gamble that ~

`gamble that ~`은 문자 그대로는 **~라고 도박하다**이지만, 실제로는 **~일 거라고 믿고 위험을 감수하다**라는 뜻이다.

```
gamble that + 문장
~라고 보고 위험을 감수하다
```

이 문장에서는:

```
she was gambling that I wouldn’t smack her
그녀는 내가 자기를 때리지 않을 거라고 보고 위험을 감수하고 있었다
```

## 6-3. wouldn’t

`wouldn’t`는 `would not`의 축약형이다.

여기서는 미래의 확신이라기보다, 그 상황에서 **내가 그렇게 하지는 않을 것이다**라는 추측/가정을 나타낸다.

```
I wouldn’t smack her
나는 그녀를 때리지는 않을 것이다
```

## 6-4. smack

`smack`은 손바닥으로 찰싹 때리다, 때리다라는 뜻이다.

```
smack someone
누군가를 찰싹 때리다
```

문맥에 따라 실제 폭력보다 과장된 말투나 코미디식 표현일 수 있다.

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| of course | 물론, 당연히 | 말하는 사람의 태도 |
| by saying that | 그렇게 말함으로써 | by + 동명사 구조 |
| gamble | 도박하다, 위험을 감수하다 | 여기서는 비유적 의미 |
| gamble that ~ | ~라고 보고 위험을 감수하다 | 명사절 that을 목적어로 받음 |
| wouldn’t | would not | ~하지 않을 것이다 |
| smack | 찰싹 때리다 | 구어체 느낌 가능 |
| her | 그녀를 | smack의 목적어 |

# 8. 자연스러운 번역

**“물론, 그렇게 말한 건 내가 자기를 때리진 않을 거라고 보고 위험을 감수한 거였지.”**

또는:

**“물론, 그렇게 말하면서 그녀는 내가 자기를 때리지 않을 거라고 믿고 배짱을 부린 거야.”**
