# Is that it?

# 문장

Is that it?

# 1. 전체 의미

`Is that it?`은 문맥에 따라 두 가지로 해석된다.

1. **그게 다야?**
2. **그게 그거야? / 그게 맞아?**

가장 흔한 뜻은 **그게 다야?**이다.

# 2. 뉘앙스

상대가 설명을 끝냈거나 무언가를 보여줬을 때, 생각보다 별것 없거나 더 있을 줄 알았을 때 쓴다.

```
Is that it?
그게 다야?
```

또 어떤 대상을 가리키며 확인할 때는 **그게 우리가 찾던 거야? / 그게 맞아?**라는 뜻이 된다.

# 3. 문장 성분 표시

평서문으로 바꾸면 다음과 같다.

```
That is it.
S    V  C
```

의문문에서는 be동사 `is`가 앞으로 온다.

```
Is that it?
V  S    C
```

# 4. 문장 형식

이 문장은 기본적으로 **2형식**이다.

```
S + V + C
That + is + it
```

# 5. 핵심 문법 포인트

`that`은 주어이고, `it`은 보어다.

`it`은 문맥에 따라 **전부, 끝, 찾던 것, 정답, 핵심** 등을 의미할 수 있다.

| 문맥 | 해석 |
| --- | --- |
| 설명이 끝났을 때 | 그게 다야? |
| 물건이나 장소를 확인할 때 | 그게 그거야? |
| 답을 확인할 때 | 그게 답이야? |

# 6. 비슷한 표현

| 표현 | 뜻 | 느낌 |
| --- | --- | --- |
| Is that it? | 그게 다야? / 그게 맞아? | 일반적인 확인 |
| That’s it? | 그게 다라고? | 놀람, 실망, 반문 느낌이 더 강함 |
| That’s it. | 그게 다야 / 바로 그거야 | 평서문 |

# 7. 자연스러운 번역

**그게 다야?**

문맥에 따라:

**그게 끝이야?**

**그게 그거야?**

**그게 맞아?**
