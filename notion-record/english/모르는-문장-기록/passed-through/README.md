# passed through

# 표현

passed through

# 1. 전체 의미

**“통과했다 / 지나갔다 / 거쳐 갔다”**

`passed through`는 기본적으로 어떤 장소, 공간, 상황, 과정 등을 **지나가다 / 통과하다 / 거쳐 가다**라는 뜻이다.

# 2. 기본 구조

```
pass through + 장소/공간/상황
~을 통과하다 / ~을 지나가다 / ~을 거쳐 가다
```

`passed`는 `pass`의 과거형 또는 과거분사형이다.

```
pass → passed → passed
```

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| passed | 지나갔다 / 통과했다 |
| through | ~을 통과하여 / ~을 지나서 |

# 4. 문장 성분 표시

`passed through`만으로는 완전한 문장이 아니라 **동사구**다.

예를 들어 문장으로 만들면:

```
He passed through the door.
S  V              O/전치사구
```

더 정확히는:

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | He | 주어 |
| V | passed | 동사 |
| 전치사구 | through the door | 문을 통과해서 |

# 5. 문장 형식

`pass through`는 보통 자동사 + 전치사구 구조로 본다.

```
S + V + 전치사구
He + passed + through the door
```

따라서 기본적으로는 **1형식 + 부사구**로 볼 수 있다.

하지만 `pass through`를 하나의 표현처럼 보면:

```
pass through + 장소
~을 통과하다
```

처럼 이해해도 된다.

# 6. 핵심 문법 포인트

## 6-1. through의 느낌

`through`는 어떤 공간의 **한쪽에서 들어가서 반대쪽으로 나오는 느낌**이다.

```
through the tunnel
터널을 통과해서
```

```
through the door
문을 통과해서
```

```
through the city
도시를 지나서
```

## 6-2. pass by와 pass through 차이

| 표현 | 의미 | 느낌 |
| --- | --- | --- |
| pass by | 옆을 지나가다 | 안으로 들어가지 않고 근처를 지나감 |
| pass through | 통과하다 / 거쳐 가다 | 안이나 중간을 지나감 |

예문:

```
He passed by the office.
그는 사무실 옆을 지나갔다.
```

```
He passed through the office.
그는 사무실을 통과해서 지나갔다.
```

# 7. 예문

```
We passed through a small town.
우리는 작은 마을을 지나갔다.
```

```
The train passed through the tunnel.
기차가 터널을 통과했다.
```

```
A strange thought passed through my mind.
이상한 생각이 내 머릿속을 스쳐 지나갔다.
```

# 8. 단어/표현 정리

| 표현 | 뜻 | 설명 |
| --- | --- | --- |
| pass | 지나가다, 통과하다 | 기본 동사 |
| passed | 지나갔다 / 통과했다 | 과거형 또는 과거분사형 |
| through | ~을 통과하여 | 안을 지나 반대쪽으로 나오는 느낌 |
| pass through | ~을 통과하다 / 거쳐 가다 | 장소, 과정, 생각 등에 사용 가능 |

# 9. 자연스러운 번역

문맥 없이 `passed through`만 보면:

**“통과했다.”**

또는:

**“지나갔다.”**

문맥에 따라:

**“거쳐 갔다.”**

라고 번역할 수 있다.
