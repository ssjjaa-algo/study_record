# I should never have let the temp touch this thing

# 문장

I should never have let the temp touch this thing.

# 1. 전체 의미

**“내가 그 임시 직원이 이걸 건드리게 하면 절대 안 됐어.”**

더 자연스럽게는:

**“그 임시 직원한테 이걸 만지게 하는 게 아니었어.”**

# 2. 뉘앙스

`should never have p.p.`는 과거에 한 행동을 강하게 후회하거나 비판할 때 쓰는 표현이다.

```
should never have done
절대 ~하지 말았어야 했다
```

말하는 사람은 이미 `the temp`가 `this thing`을 만졌고, 그 결과가 좋지 않다고 느끼는 상황이다.

`let`은 사역동사로, 여기서는 **~하게 두다 / 허락하다 / 하게 만들다**라는 뜻이다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| I | 나는 |
| should never have let | 절대 ~하게 두지 말았어야 했다 |
| the temp | 그 임시 직원이 |
| touch this thing | 이걸 만지게 / 건드리게 |

# 4. 문장 성분 표시

```
I should never have let the temp touch this thing.
S V                    O        OC
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | I | 주어 |
| V | should never have let | 조동사 완료형 + 사역동사 |
| O | the temp | 목적어, 행동하는 사람 |
| OC | touch this thing | 목적격보어, the temp가 한 행동 |

`touch this thing` 내부:

```
touch this thing
V     O
```

# 5. 문장 형식

이 문장은 **5형식**이다.

```
S + V + O + OC
I + should never have let + the temp + touch this thing
```

`let`은 사역동사이므로 뒤에 목적어와 동사원형이 온다.

```
let + 목적어 + 동사원형
~가 ~하게 두다 / 허락하다
```

# 6. 핵심 문법 포인트

## 6-1. should never have p.p.

```
should never have + 과거분사
절대 ~하지 말았어야 했다
```

예문:

```
I should never have said that.
그 말을 절대 하지 말았어야 했어.
```

```
You should never have trusted him.
너는 그를 절대 믿지 말았어야 했어.
```

이 문장에서는 `let`의 과거분사도 `let`이다.

```
let - let - let
```

그래서:

```
should never have let
절대 하게 두지 말았어야 했다
```

## 6-2. let + 목적어 + 동사원형

`let`은 대표적인 사역동사다.

```
let + O + 동사원형
O가 ~하게 하다 / O가 ~하도록 허락하다
```

이 문장에서는:

```
let the temp touch this thing
임시 직원이 이걸 만지게 두다
```

`touch`가 동사원형인 이유는 `let` 뒤의 목적격보어는 원형부정사를 쓰기 때문이다.

## 6-3. the temp

`temp`는 `temporary worker/employee`의 줄임말이다.

```
the temp
그 임시 직원 / 임시 알바 / 단기 직원
```

문맥에 따라 사무실의 임시직 직원을 뜻한다.

## 6-4. touch this thing

`touch`는 단순히 손으로 만지다뿐 아니라, 기계나 중요한 물건을 **건드리다 / 손대다**라는 뜻으로도 쓰인다.

```
Don’t touch this thing.
이거 건드리지 마.
```

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| should never have p.p. | 절대 ~하지 말았어야 했다 | 과거 행동에 대한 강한 후회 |
| let | ~하게 두다 / 허락하다 | 사역동사 |
| the temp | 임시 직원 | temporary worker의 줄임말 |
| touch | 만지다 / 건드리다 | 기계나 물건에 손대다 |
| this thing | 이거 / 이 물건 | 말하는 사람이 가리키는 대상 |

# 8. 자연스러운 번역

**“그 임시 직원한테 이걸 만지게 하는 게 아니었어.”**

또는:

**“내가 그 임시 직원이 이걸 건드리게 놔두면 절대 안 됐어.”**

문맥에 따라:

**“아, 그 임시 직원한테 이걸 맡기는 게 아니었는데.”**
