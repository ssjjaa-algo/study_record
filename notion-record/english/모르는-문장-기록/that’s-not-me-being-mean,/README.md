# That’s not me being mean,

# 문장

That’s not me being mean,

# 1. 전체 의미

**“그건 내가 못되게 구는 게 아니야.”**

또는 더 자연스럽게:

**“내가 일부러 못되게 말하는 건 아니야.”**

상대가 자신의 말이나 행동을 나쁘게 받아들일 수 있는 상황에서, 말하는 사람이 **나는 악의적으로 그러는 게 아니다**라고 방어하거나 설명하는 표현이다.

# 2. 뉘앙스

`that’s not ~`은 **그건 ~가 아니다**라는 뜻이다.

`me being mean`은 직역하면 **내가 못되게 굴고 있는 것**이다.

즉 전체적으로는:

```
그건 내가 못되게 행동하는 게 아니야.
그렇게 보일 수 있지만 악의는 없어.
```

라는 느낌이다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| That’s not | 그건 ~가 아니다 |
| me | 나 / 내가 |
| being mean | 못되게 구는 것 / 못되게 행동하는 것 |

# 4. 문장 성분 표시

```
That’s not me being mean.
S     V   C
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | That | 주어, 앞에서 말한 상황/행동 |
| V | is not | be동사 부정형 |
| C | me being mean | 보어, 그게 무엇이 아닌지 설명 |

`me being mean` 내부 구조:

```
me being mean
의미상 주어 + 동명사구
```

| 부분 | 역할 |
| --- | --- |
| me | 의미상 주어, 못되게 구는 사람 |
| being mean | 못되게 구는 것 |

# 5. 문장 형식

이 문장은 **2형식**이다.

```
S + V + C
That + is not + me being mean
```

해석 구조는:

```
That = me being mean 이 아니다
```

즉:

```
그 상황은 내가 못되게 구는 것이 아니다
```

# 6. 핵심 문법 포인트

## 6-1. me being mean 구조

`me being mean`은 **목적격 + 동명사** 구조다.

```
me + being mean
내가 못되게 구는 것
```

문법적으로 더 격식 있게는 `my being mean`도 가능하다.

```
That’s not my being mean.
```

하지만 일상 회화에서는 `me being mean`이 훨씬 자연스럽다.

## 6-2. being + 형용사

```
being + 형용사
~하게 굴고 있는 것 / ~한 태도를 보이는 것
```

예문:

```
He is being rude.
그는 무례하게 굴고 있어.
```

```
You’re being too sensitive.
너 너무 예민하게 구는 거야.
```

`He is rude`는 성격 자체가 무례하다는 느낌이고, `He is being rude`는 지금 행동이 무례하다는 느낌이다.

## 6-3. mean

`mean`은 동사로는 **의미하다**이지만, 형용사로는 **못된, 심술궂은, 불친절한**이라는 뜻이다.

```
You’re mean.
너 못됐어.
```

```
I’m not being mean.
나 못되게 구는 거 아니야.
```

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| That’s not ~ | 그건 ~가 아니다 | 앞의 상황을 부정 |
| me being ~ | 내가 ~하는 것 | 회화식 동명사 구조 |
| being mean | 못되게 구는 것 | 일시적인 행동/태도 |
| mean | 못된, 심술궂은 | 형용사 의미 |

# 8. 자연스러운 번역

**“내가 못되게 구는 건 아니야.”**

또는:

**“이건 내가 일부러 못되게 말하는 게 아니야.”**

문맥에 따라:

**“나 악의로 그러는 거 아니야.”**
