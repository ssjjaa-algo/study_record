# You should know me better than that

# 문장

You should know me better than that.

# 1. 전체 의미

**“너라면 나를 그 정도보다는 더 잘 알아야지.”**

또는 더 자연스럽게:

**“네가 나를 안다면, 내가 그런 사람은 아니라는 걸 알아야지.”**

상대가 나를 오해했거나, 내가 하지 않을 만한 일을 했다고 생각했을 때 쓰는 표현이다.

# 2. 뉘앙스

`You should know me better than that`은 단순히 “너는 나를 더 잘 알아야 한다”가 아니라, 상대의 판단이나 말에 대한 **서운함 / 실망 / 가벼운 질책**이 들어간 표현이다.

즉 말하는 사람의 느낌은 다음과 같다.

```
너는 나를 잘 알잖아.
그런 식으로 나를 생각하면 안 되지.
내가 그런 사람은 아니라는 걸 알아야지.
```

여기서 `that`은 앞에서 나온 상대의 생각, 말, 오해, 행동을 가리킨다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| You | 너는 |
| should know | 알아야 한다 |
| me | 나를 |
| better than that | 그것보다는 더 잘 / 그런 식으로 생각할 정도는 아니게 |

# 4. 문장 성분 표시

```
You should know me better than that.
S   V           O  Adv/비교구
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | You | 주어 |
| V | should know | 조동사 should + 동사원형 know |
| O | me | 목적어, 아는 대상 |
| 부사구 | better than that | 얼마나 잘 알아야 하는지를 설명하는 비교 표현 |

# 5. 문장 형식

이 문장은 기본적으로 **3형식**이다.

```
S + V + O
You + should know + me
```

뒤의 `better than that`은 `know`를 꾸며주는 비교 부사구다.

전체 구조는 다음과 같다.

```
You should know me / better than that.
너는 나를 알아야 한다 / 그것보다는 더 잘.
```

# 6. 핵심 문법 포인트

## 6-1. should know

`should + 동사원형`은 **~해야 한다 / ~하는 게 당연하다**라는 뜻이다.

```
You should know.
너는 알아야 해.
```

이 문장에서는 단순 의무보다 **상대라면 당연히 알아야 한다**는 기대가 들어 있다.

## 6-2. know someone better than that

```
know someone better than that
누군가를 그 정도보다는 더 잘 알다
```

이 표현은 관용적으로 **그 사람이 그런 식으로 행동할 사람이 아니라는 걸 알다**라는 뜻으로 자주 쓰인다.

예문:

```
You should know me better than that.
너라면 내가 그런 사람은 아니라는 걸 알아야지.
```

```
I thought you knew me better than that.
네가 나를 그 정도보다는 더 잘 안다고 생각했어.
```

## 6-3. than that

`than that`의 `that`은 앞에서 나온 말이나 상황을 가리킨다.

예를 들어 상대가 “네가 거짓말했지?”라고 의심했다면:

```
You should know me better than that.
너라면 내가 거짓말할 사람이 아니라는 걸 알아야지.
```

즉 `that`은 **나를 그렇게 보는 것 / 그런 식으로 판단하는 것**을 의미한다.

## 6-4. 생략된 의미

이 문장은 다음과 같이 풀어볼 수 있다.

```
You should know me better than to think that.
너는 그런 생각을 할 정도보다는 나를 더 잘 알아야 한다.
```

또는:

```
You should know me well enough not to think that.
너는 그런 생각을 하지 않을 만큼 나를 잘 알아야 한다.
```

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| should | ~해야 한다 | 기대, 당위, 가벼운 질책 |
| know | 알다 | 사람의 성격이나 성향을 안다는 의미 |
| know me | 나를 알다 | 여기서는 내 성격을 안다는 뜻 |
| better than that | 그것보다는 더 잘 | 상대의 오해를 지적하는 비교 표현 |
| that | 그것, 그런 생각 | 앞의 말이나 상황을 가리킴 |

# 8. 자연스러운 번역

**“너라면 나를 그 정도보다는 더 잘 알아야지.”**

또는:

**“네가 나를 안다면, 내가 그런 사람은 아니라는 걸 알아야지.”**

문맥에 따라:

**“나를 그렇게밖에 모른다고?”**
