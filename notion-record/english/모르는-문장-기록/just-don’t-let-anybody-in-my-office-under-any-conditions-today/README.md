# Just don’t let anybody in my office under any conditions today.

# 1. 전체 의미

**“오늘은 어떤 경우에도 아무도 내 사무실에 들여보내지 마.”**

강한 금지 명령이다. `under any conditions`가 들어가서 **예외 없이, 무슨 일이 있어도**라는 느낌이 강해진다.

# 2. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| Just don’t let | 그냥 ~하게 하지 마 / 절대 ~하지 마 |
| anybody | 아무도 |
| in my office | 내 사무실 안으로 |
| under any conditions | 어떤 조건에서도 / 어떤 경우에도 |
| today | 오늘 |

# 3. 문법 구조

| 요소 | 내용 | 설명 |
| --- | --- | --- |
| 명령문 | Don’t let ~ | 부정 명령문 |
| 동사 | let | 사역동사 |
| 목적어 | anybody | 들여보내면 안 되는 대상 |
| 목적격 보어 | in my office | anybody의 상태/위치 |
| 부사구 | under any conditions | 조건을 나타냄 |
| 부사 | today | 시간 표현 |

구조로 보면:

```
Don’t let + 목적어 + 보어
Don’t let + anybody + in my office
```

즉,

```
누구도 / 내 사무실 안에 있게 하지 마
```

라는 구조다.

# 4. 핵심 문법 포인트

`let`은 사역동사라서 보통 뒤에 **목적어 + 동사원형**이 온다.

예:

```
Don’t let him enter.
그가 들어오게 하지 마.
```

그런데 여기서는 `enter` 같은 동사가 생략된 느낌이고, `in my office`가 **상태/위치 보어**처럼 쓰였다.

```
Don’t let anybody in my office.
아무도 내 사무실 안에 들이지 마.
```

여기서 `in`은 동사가 아니라 **전치사**다. 따라서 `in my office`는 전치사구이고, `anybody`가 어디에 있게 되는지를 설명한다.

# 5. 단어 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| just | 그냥, 제발, 하여튼 | 여기서는 명령을 강조 |
| don’t let | ~하게 하지 마 | 부정 명령 |
| anybody | 아무도 | 부정문에서 “누구도” |
| office | 사무실 | my office = 내 사무실 |
| under any conditions | 어떤 조건에서도 | 강한 예외 없음 표현 |
| today | 오늘 | 시간 부사 |
