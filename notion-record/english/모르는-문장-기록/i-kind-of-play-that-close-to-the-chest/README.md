# I kind of play that close to the chest

# 문장

I kind of play that close to the chest.

# 1. 전체 의미

**“나는 그건 좀 숨겨두는 편이야.”**

또는 더 자연스럽게:

**“그건 좀 남에게 잘 말 안 해.”**

문맥에 따라:

**“그건 내가 좀 비밀로 해두는 편이야.”**

# 2. 뉘앙스

`play something close to the chest`는 관용표현이다.

원래는 카드 게임에서 자기 카드를 가슴 가까이에 붙여 들고, 남들이 보지 못하게 하는 이미지다. 그래서 실제 의미는 **정보, 감정, 계획 등을 드러내지 않고 숨기다 / 신중하게 감추다**이다.

`kind of`가 있어서 강하게 “절대 말 안 한다”가 아니라, **좀 그런 편이다 / 약간 조심스럽게 숨긴다**는 부드러운 느낌이다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| I | 나는 |
| kind of | 좀 / 약간 / 어느 정도 |
| play that | 그것을 다룬다 / 처리한다 |
| close to the chest | 가슴 가까이에, 즉 남에게 드러내지 않고 |

# 4. 문장 성분 표시

```
I kind of play that close to the chest.
S          V    O    OC/상태 설명
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | I | 주어 |
| 부사 | kind of | 정도를 부드럽게 함 |
| V | play | 여기서는 관용표현의 동사 |
| O | that | 목적어, 숨기거나 조심스럽게 다루는 내용 |
| OC/부사적 표현 | close to the chest | 목적어 that을 어떻게 다루는지 설명 |

문법적으로는 관용표현 전체를 하나의 덩어리로 보는 것이 가장 자연스럽다.

```
play something close to the chest
무언가를 숨기다 / 남에게 잘 드러내지 않다
```

# 5. 문장 형식

이 문장은 의미상 **5형식처럼** 볼 수 있다.

```
S + V + O + OC
I + play + that + close to the chest
```

다만 `play that close to the chest`는 관용표현이므로, 실제 이해는 다음 덩어리로 하는 것이 좋다.

```
play that close to the chest
그것을 숨기다 / 조심스럽게 감추다
```

# 6. 핵심 문법 포인트

## 6-1. kind of

`kind of`는 회화에서 **좀 / 약간 / 어느 정도**라는 뜻이다.

```
I kind of like it.
나 그거 좀 좋아해.
```

```
I’m kind of tired.
나 좀 피곤해.
```

이 문장에서는 말하는 사람이 자신의 태도를 부드럽게 표현한다.

```
I kind of play that close to the chest.
나는 그건 좀 숨기는 편이야.
```

## 6-2. play something close to the chest

```
play something close to the chest
무언가를 숨기다 / 조심스럽게 감추다 / 남에게 잘 드러내지 않다
```

주로 감정, 계획, 정보, 의도 등을 쉽게 말하지 않을 때 쓴다.

예문:

```
He plays his plans close to the chest.
그는 자기 계획을 잘 드러내지 않는다.
```

```
She tends to play her feelings close to the chest.
그녀는 자기 감정을 잘 드러내지 않는 편이다.
```

```
I play my cards close to the chest.
나는 내 패를 잘 드러내지 않는다.
```

## 6-3. that

`that`은 앞에서 말한 어떤 정보, 감정, 계획, 생각 등을 가리킨다.

예를 들어 누가 “너 그 계획 누구한테 말했어?”라고 물었을 때:

```
I kind of play that close to the chest.
그건 내가 좀 숨겨두는 편이야.
```

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| kind of | 좀, 약간 | 말을 부드럽게 함 |
| play | 다루다, 행동하다 | 여기서는 관용표현 일부 |
| that | 그것 | 앞에서 나온 내용 |
| close to the chest | 가슴 가까이에 | 비유적으로 남에게 보이지 않게 |
| play something close to the chest | 무언가를 숨기다 / 신중히 감추다 | 계획, 감정, 정보 등을 잘 드러내지 않음 |

# 8. 자연스러운 번역

**“그건 내가 좀 숨겨두는 편이야.”**

또는:

**“그건 남한테 잘 말 안 해.”**

문맥에 따라:

**“나는 그런 건 좀 조심스럽게 감춰두는 편이야.”**
