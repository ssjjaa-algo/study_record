# Someone in this office is coming up with all this ridiculous stuff.

# 문장

Someone in this office is coming up with all this ridiculous stuff.

# 1. 전체 의미

**“이 사무실의 누군가가 이 말도 안 되는 것들을 다 생각해내고 있어.”**

또는 더 자연스럽게:

**“이 사무실 안의 누군가가 이런 터무니없는 짓들을 계속 만들어내고 있어.”**

# 2. 뉘앙스

`someone in this office`는 **이 사무실 안의 누군가**라는 뜻이다. 누군지는 모르거나, 일부러 특정하지 않고 말하는 느낌이다.

`come up with`는 **생각해내다, 제안하다, 만들어내다**라는 뜻의 구동사다.

`all this ridiculous stuff`는 **이 모든 말도 안 되는 것들 / 터무니없는 일들**이라는 뜻이다. 말하는 사람이 그 아이디어나 행동을 부정적으로 보고 있다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| Someone | 누군가가 |
| in this office | 이 사무실 안의 |
| is coming up with | 생각해내고 있다 / 만들어내고 있다 |
| all this ridiculous stuff | 이 모든 말도 안 되는 것들 |

# 4. 문장 성분 표시

```
Someone in this office is coming up with all this ridiculous stuff.
S                       V                  O
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | Someone in this office | 주어, 이 사무실의 누군가 |
| V | is coming up with | 동사구, 생각해내고 있다 |
| O | all this ridiculous stuff | 목적어, 말도 안 되는 것들 |

# 5. 문장 형식

이 문장은 크게 보면 **3형식**이다.

```
S + V + O
Someone in this office + is coming up with + all this ridiculous stuff
```

해석 구조는:

```
S가 O를 생각해내고 있다 / 만들어내고 있다
```

즉:

```
이 사무실의 누군가가 + 이 말도 안 되는 것들을 + 생각해내고 있다
```

# 6. 문법 구조

## 6-1. Someone in this office

```
Someone + in this office
```

`in this office`는 앞의 `someone`을 설명하는 전치사구다.

```
someone in this office
이 사무실 안에 있는 누군가
```

## 6-2. is coming up with

```
is + coming up with
```

현재진행형이다.

```
be동사 + -ing
```

여기서는 **지금 계속 그런 아이디어를 내고 있다 / 만들어내고 있다**는 느낌이다.

## 6-3. come up with

`come up with`는 하나의 덩어리로 봐야 한다.

```
come up with + 명사
~을 생각해내다 / 제안하다 / 마련하다
```

예문:

```
She came up with a good idea.
그녀는 좋은 아이디어를 생각해냈다.
```

# 7. 핵심 문법 포인트

## 7-1. come up with는 구동사인가?

`come up with`는 넓게 보면 구동사 또는 multi-word verb로 볼 수 있다.

하지만 `write down`, `pick up`처럼 자유롭게 분리되는 구동사는 아니다.

```
come up with an idea
```

은 가능하지만,

```
come an idea up with
```

은 불가능하다.

즉 `come up with`는 통째로 외워야 하는 표현이다.

## 7-2. all this ridiculous stuff

```
all + this + ridiculous + stuff
```

| 단어 | 역할 |
| --- | --- |
| all | 전부, 모든 |
| this | 이 |
| ridiculous | 말도 안 되는, 터무니없는 |
| stuff | 것들, 일들 |

`stuff`는 셀 수 없는 명사처럼 쓰이는 구어체 표현이다. 여러 물건, 일, 내용 등을 뭉뚱그려 말할 때 쓴다.

# 8. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| someone | 누군가 | 특정하지 않은 사람 |
| in this office | 이 사무실 안의 | someone을 설명 |
| come up with | 생각해내다, 제안하다, 마련하다 | 통째로 외우는 표현 |
| ridiculous | 말도 안 되는, 터무니없는 | 부정적 뉘앙스 |
| stuff | 것들, 일들 | 구어체 표현 |
| all this stuff | 이 모든 것들 | 뭉뚱그려 말하는 표현 |

# 9. 자연스러운 번역

**“이 사무실의 누군가가 이런 말도 안 되는 것들을 다 생각해내고 있어.”**

또는:

**“이 사무실 안에 누군가가 이런 터무니없는 짓들을 계속 만들어내고 있어.”**
