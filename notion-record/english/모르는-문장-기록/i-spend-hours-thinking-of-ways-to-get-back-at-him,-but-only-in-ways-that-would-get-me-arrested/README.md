# I spend hours thinking of ways to get back at him, but only in ways that would get me arrested

# 문장

I spend hours thinking of ways to get back at him, but only in ways that would get me arrested.

# 1. 전체 의미

**“나는 그에게 복수할 방법들을 생각하느라 몇 시간을 보내는데, 문제는 그 방법들이 다 내가 체포될 만한 것들이라는 거야.”**

더 자연스럽게는:

**“걔한테 복수할 방법을 몇 시간씩 생각하긴 하는데, 죄다 내가 체포될 만한 방법들뿐이야.”**

# 2. 뉘앙스

`spend hours thinking of ~`는 **~을 생각하느라 몇 시간을 쓰다**라는 뜻이다.

`get back at him`은 **그에게 복수하다 / 앙갚음하다**라는 구어체 표현이다.

`but only in ways that would get me arrested`는 **하지만 그 방법들이라곤 내가 체포될 만한 방법들뿐이다**라는 의미다.

전체적으로는 분노나 복수심을 말하지만, `would get me arrested` 때문에 과장되고 코믹한 느낌도 있다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| I spend hours | 나는 몇 시간을 보낸다 |
| thinking of ways | 방법들을 생각하면서 |
| to get back at him | 그에게 복수할 / 앙갚음할 |
| but only in ways | 하지만 오직 그런 방식들로만 |
| that would get me arrested | 나를 체포되게 만들 만한 |

# 4. 문장 성분 표시

```
I spend hours thinking of ways to get back at him, but only in ways that would get me arrested.
S V     O     Adv/V-ing phrase
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | I | 주어 |
| V | spend | 동사 |
| O | hours | 목적어, 쓰는 시간 |
| 부사/분사구 | thinking of ways to get back at him | 무엇을 하며 시간을 보내는지 설명 |
| 부사구 | but only in ways that would get me arrested | 어떤 방식인지 추가 설명 |

`that would get me arrested` 내부:

```
that would get me arrested
S    V         O  OC
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | that | 관계대명사, ways를 가리킴 |
| V | would get | ~하게 만들 것이다 |
| O | me | 목적어 |
| OC | arrested | 목적격 보어, 내가 처하게 되는 상태 |

# 5. 문장 형식

주절은 **3형식**이다.

```
S + V + O
I + spend + hours
```

`thinking of ways to get back at him`은 시간을 어떻게 보내는지 설명하는 분사구다.

관계절 `that would get me arrested`는 **5형식**이다.

```
S + V + O + OC
that + would get + me + arrested
```

해석 구조는:

```
그 방법들이 나를 체포된 상태가 되게 만들 것이다
```

# 6. 핵심 문법 포인트

## 6-1. spend time V-ing

```
spend + 시간 + V-ing
~하느라 시간을 보내다
```

이 문장에서는:

```
spend hours thinking of ways
방법들을 생각하느라 몇 시간을 보내다
```

예문:

```
I spent all night studying.
나는 공부하느라 밤새 시간을 보냈다.
```

## 6-2. think of ways to V

```
think of ways to V
~할 방법들을 생각하다
```

이 문장에서는:

```
ways to get back at him
그에게 복수할 방법들
```

## 6-3. get back at someone

`get back at someone`은 **~에게 복수하다 / 앙갚음하다**라는 뜻의 구어체 표현이다.

```
I’ll get back at him.
그에게 복수할 거야.
```

`get back to someone`과 헷갈리면 안 된다.

| 표현 | 뜻 |
| --- | --- |
| get back at someone | ~에게 복수하다 |
| get back to someone | ~에게 다시 연락하다 |

## 6-4. would get me arrested

```
get + O + p.p.
O가 ~된 상태가 되게 하다
```

이 문장에서는:

```
get me arrested
나를 체포되게 만들다 / 내가 체포되게 하다
```

`arrested`는 과거분사로, `me`가 처하게 되는 상태를 설명한다.

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| spend hours | 몇 시간을 보내다 | 시간 소비 표현 |
| think of | ~을 생각하다 | 아이디어를 떠올리다 |
| ways to V | ~할 방법들 | to부정사가 ways를 설명 |
| get back at someone | ~에게 복수하다 | 구어체 표현 |
| but only | 하지만 오직 ~뿐 | 제한 강조 |
| would | ~할 것이다 / ~할 법하다 | 가정적 가능성 |
| get me arrested | 나를 체포되게 하다 | get + O + p.p. |
| arrested | 체포된 | 과거분사 |

# 8. 자연스러운 번역

**“걔한테 복수할 방법을 몇 시간씩 생각하긴 하는데, 죄다 내가 체포될 만한 방법들뿐이야.”**

또는:

**“그 사람한테 앙갚음할 방법을 한참 생각하지만, 떠오르는 건 전부 내가 잡혀갈 만한 것들뿐이야.”**
