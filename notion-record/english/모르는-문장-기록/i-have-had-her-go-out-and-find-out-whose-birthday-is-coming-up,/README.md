# I have had her go out and find out whose birthday is coming up,

# 문장

I have had her go out and find out whose birthday is coming up,

# 1. 전체 의미

**“나는 그녀에게 밖에 나가서 누구 생일이 다가오고 있는지 알아보게 했어.”**

또는 더 자연스럽게:

**“내가 그녀한테 나가서 곧 생일인 사람이 누구인지 알아보라고 시켰어.”**

# 2. 뉘앙스

`have had her go out and find out ~`는 **누군가에게 어떤 일을 하게 했다**는 뜻이다.

여기서 `have had`는 현재완료 형태다.

```
have had + 사람 + 동사원형
```

즉, 과거 어느 시점부터 지금까지 이어지는 맥락에서 **그녀에게 그 일을 시켜두었다 / 하게 해왔다**는 느낌이 있다.

`find out`은 단순히 `find`처럼 물건을 찾는 게 아니라, **정보를 알아내다**라는 뜻이다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| I | 나는 |
| have had her | 그녀에게 ~하게 해왔다 / 시켜두었다 |
| go out | 밖에 나가다 |
| and find out | 그리고 알아내다 |
| whose birthday | 누구의 생일이 |
| is coming up | 다가오고 있다 |

# 4. 문장 성분 표시

```
I have had her go out and find out whose birthday is coming up.
S    V      O   OC
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | I | 주어 |
| V | have had | 사역동사 have의 현재완료형 |
| O | her | 목적어, 일을 하게 된 사람 |
| OC | go out and find out whose birthday is coming up | 목적격 보어, her가 하는 행동 |

더 세부적으로 보면:

```
her + go out + and + find out [whose birthday is coming up]
O     OC1            OC2
```

`whose birthday is coming up`은 `find out`의 목적어 역할을 하는 간접의문문이다.

# 5. 문장 형식

이 문장은 **5형식**으로 볼 수 있다.

```
S + V + O + OC
I + have had + her + go out and find out whose birthday is coming up
```

해석 구조는:

```
S가 O에게 OC하게 하다
```

즉:

```
나는 그녀가 밖에 나가서 누구 생일이 다가오는지 알아보게 했다
```

# 6. 핵심 문법 포인트

## 6-1. have + 목적어 + 동사원형

`have`는 사역동사로 쓰일 수 있다.

```
have + O + 동사원형
O에게 ~하게 하다 / 시키다
```

예문:

```
I had him check the report.
나는 그에게 보고서를 확인하게 했다.
```

이 문장에서는 현재완료가 붙어서:

```
I have had her go out...
나는 그녀에게 나가게 해왔다 / 나가게 했다
```

가 된다.

## 6-2. have had 구조

`have had`는 `have`가 두 번 나온 것처럼 보여서 헷갈리지만 역할이 다르다.

```
I have had her go out.
```

| 단어 | 역할 |
| --- | --- |
| have | 현재완료를 만드는 조동사 |
| had | 사역동사 have의 과거분사 |

즉 구조는:

```
have + p.p.
have + had
```

현재완료다.

## 6-3. go out and find out

```
go out and find out
밖에 나가서 알아내다
```

`go out`과 `find out`이 `and`로 연결되어 있다.

둘 다 `her`가 하는 행동이다.

## 6-4. find out

`find out`은 **정보를 알아내다**라는 뜻이다.

```
find out + 정보
~을 알아내다
```

예문:

```
Find out who called.
누가 전화했는지 알아봐.
```

```
I found out the truth.
나는 진실을 알아냈다.
```

## 6-5. whose birthday is coming up

```
whose birthday is coming up
누구의 생일이 다가오고 있는지
```

이 부분은 간접의문문이다.

직접의문문이면:

```
Whose birthday is coming up?
누구 생일이 다가오고 있어?
```

하지만 `find out` 뒤에 들어가면 명사절처럼 쓰인다.

```
find out whose birthday is coming up
누구 생일이 다가오는지 알아내다
```

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| have someone do | 누군가에게 ~하게 하다 | 사역동사 구조 |
| have had | 현재완료 + 사역동사 have | 시켜두었다 / 하게 해왔다 |
| her | 그녀에게 / 그녀가 | 목적어 |
| go out | 밖에 나가다 | 장소 밖으로 이동 |
| find out | 알아내다 | 정보 파악 |
| whose | 누구의 | 소유를 묻는 의문사 |
| birthday | 생일 | whose와 함께 사용 |
| come up | 다가오다, 예정되어 있다 | 일정/행사에 자주 사용 |

# 8. 자연스러운 번역

**“내가 그녀한테 밖에 나가서 곧 생일인 사람이 누구인지 알아보라고 시켰어.”**

또는:

**“나는 그녀에게 누가 곧 생일인지 알아보게 해뒀어.”**
