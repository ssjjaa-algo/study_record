# I should never have let the temp touch this thing

# 문장

I should never have let the temp touch this thing.

# 1. 전체 의미

**“내가 임시 직원이 이걸 만지게 하면 절대 안 됐는데.”**

또는 더 자연스럽게:

**“임시 직원한테 이걸 건드리게 하는 게 아니었어.”**

# 2. 뉘앙스

`should never have p.p.`는 과거에 한 일을 후회하거나 자책할 때 쓰는 표현이다.

이 문장에서는 말하는 사람이 과거에 `the temp`에게 `this thing`을 만지도록 허락했거나 방치했고, 그 결과 문제가 생겨서 후회하는 느낌이다.

```
내가 그 사람한테 이걸 건드리게 하면 안 됐어.
괜히 맡겼어.
이건 내 실수였어.
```

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| I | 나는 |
| should never have let | 절대 ~하게 두지 말았어야 했다 |
| the temp | 그 임시 직원 / 임시직 직원 |
| touch this thing | 이걸 만지다 / 건드리다 |

# 4. 문장 성분 표시

```
I should never have let the temp touch this thing.
S V                    O        OC
```

| 성분 | 단어/구 | 역할 |
| --- | --- | --- |
| S | I | 주어 |
| V | should never have let | 조동사 완료 구조 + 사역동사 let |
| O | the temp | 목적어, 행동을 하는 사람 |
| OC | touch this thing | 목적격보어, 임시 직원이 한 행동 |

`touch this thing` 내부 구조:

```
touch this thing
V     O
```

# 5. 문장 형식

이 문장은 **5형식**이다.

```
S + V + O + OC
I + should never have let + the temp + touch this thing
```

`let`은 사역동사라서 다음 구조를 만든다.

```
let + 목적어 + 동사원형
~가 ...하게 두다 / 허락하다
```

따라서:

```
let the temp touch this thing
임시 직원이 이걸 만지게 하다 / 만지도록 두다
```

# 6. 핵심 문법 포인트

## 6-1. should never have p.p.

```
should never have + p.p.
절대 ~하지 말았어야 했다
```

과거에 실제로 한 일을 후회할 때 쓴다.

```
I should never have said that.
내가 그 말을 절대 하지 말았어야 했어.
```

```
You should never have trusted him.
너는 그를 절대 믿지 말았어야 했어.
```

이 문장에서는:

```
I should never have let...
내가 절대 ~하게 두지 말았어야 했다
```

## 6-2. let + 목적어 + 동사원형

`let`은 사역동사 중 하나다.

```
let + O + 동사원형
O가 ~하게 하다 / 허락하다 / 내버려 두다
```

예문:

```
Let him go.
그를 가게 둬.
```

```
I let her use my phone.
나는 그녀가 내 휴대폰을 쓰게 해줬다.
```

이 문장에서는:

```
let the temp touch this thing
임시 직원이 이걸 만지게 하다
```

## 6-3. the temp

`temp`는 `temporary worker` 또는 `temporary employee`의 줄임말이다.

```
the temp
그 임시 직원 / 임시직 직원
```

예문:

```
The temp is helping us this week.
그 임시 직원이 이번 주에 우리를 도와주고 있어.
```

## 6-4. touch this thing

`touch`는 단순히 손으로 만지는 뜻도 있지만, 문맥에 따라 **건드리다 / 조작하다 / 손대다**라는 뜻도 된다.

```
Don’t touch this thing.
이거 건드리지 마.
```

# 7. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| should have p.p. | ~했어야 했다 | 과거에 하지 않은 일에 대한 후회 |
| should never have p.p. | 절대 ~하지 말았어야 했다 | 과거 행동에 대한 강한 후회 |
| let | ~하게 두다 / 허락하다 | 사역동사 |
| temp | 임시 직원 | temporary worker/employee의 줄임말 |
| touch | 만지다 / 건드리다 | 기계나 물건을 조작한다는 의미 가능 |
| this thing | 이거 / 이 물건 | 말하는 사람이 가리키는 대상 |

# 8. 자연스러운 번역

**“임시 직원한테 이걸 건드리게 하는 게 아니었어.”**

또는:

**“내가 그 임시 직원이 이걸 만지게 놔두면 절대 안 됐는데.”**

문맥에 따라:

**“아, 임시 직원한테 이걸 맡기는 게 아니었어.”**
