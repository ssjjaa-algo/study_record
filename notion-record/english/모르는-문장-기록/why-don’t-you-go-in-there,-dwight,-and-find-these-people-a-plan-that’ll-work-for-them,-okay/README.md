# Why don’t you go in there, Dwight, and find these people a plan that’ll work for them, okay?

# 문장

Why don’t you go in there, Dwight, and find these people a plan that’ll work for them, okay?

# 1. 전체 의미

**“드와이트, 저 안에 들어가서 이 사람들에게 맞는, 효과가 있을 만한 계획을 하나 찾아줘, 알겠지?”**

상대에게 어떤 행동을 하라고 부드럽게 제안하거나 지시하는 문장이다. 겉으로는 질문 형태지만 실제 의미는 명령이나 요청에 가깝다.

# 2. 뉘앙스

`Why don’t you ~?`는 직역하면 “왜 ~하지 않니?”이지만, 실제 회화에서는 **“~하는 게 어때?”**, **“~해봐”**라는 제안/권유 표현이다.

여기서는 `Dwight`에게 직접 말하고 있고, 뒤에 `okay?`가 붙어 있어서 상대의 동의를 확인하는 동시에 지시를 부드럽게 마무리하는 느낌이다.

`these people`은 말하는 사람과 듣는 사람 앞에 있는 특정 사람들을 가리킨다. 약간 거리감을 두고 “이 사람들”이라고 부르는 표현이다.

`a plan that’ll work for them`은 “그들에게 효과가 있을 계획”, 즉 그 사람들의 상황에 맞는 해결책을 의미한다.

# 3. 구문 나누기

| 구문 | 의미 |
| --- | --- |
| Why don’t you | ~하는 게 어때? / ~해봐 |
| go in there | 저 안으로 들어가다 |
| Dwight | 드와이트, 부르는 말 |
| and find | 그리고 찾아라 / 찾아봐라 |
| these people | 이 사람들에게 |
| a plan | 계획 하나를 |
| that’ll work for them | 그들에게 효과가 있을 / 그들에게 맞을 |
| okay? | 알겠지? / 괜찮지? |

# 4. 문법 구조

| 요소 | 내용 | 설명 |
| --- | --- | --- |
| 제안 표현 | Why don’t you ~? | 질문 형태의 제안/지시 |
| 주어 | you | 듣는 사람, Dwight |
| 동사 1 | go | 가다 |
| 부사구 | in there | 저 안으로 |
| 호격 | Dwight | 상대방 이름을 부르는 삽입 표현 |
| 접속사 | and | 두 동작을 연결 |
| 동사 2 | find | 찾다 |
| 간접목적어 | these people | 이 사람들에게 |
| 직접목적어 | a plan | 계획 하나를 |
| 관계절 | that’ll work for them | plan을 설명하는 절 |
| 확인 표현 | okay? | 동의/확인 |

구조로 보면:

```
Why don’t you + 동사원형 A + and + 동사원형 B ...?
```

이 문장에서는:

```
Why don’t you go in there and find these people a plan?
```

즉,

```
네가 저 안에 들어가서 + 이 사람들에게 계획을 찾아주는 게 어때?
```

라는 구조다.

# 5. 핵심 문법 포인트

## 5-1. Why don’t you ~?

`Why don’t you ~?`는 문자 그대로는 “왜 너는 ~하지 않니?”이지만, 회화에서는 자주 **제안**으로 쓰인다.

```
Why don’t you sit down?
앉는 게 어때? / 앉아.
```

상황에 따라 부드러운 제안이 될 수도 있고, 사실상 지시가 될 수도 있다.

## 5-2. go in there

`go in there`는 “저 안으로 들어가다”라는 뜻이다.

```
go + in there
가다 + 저 안으로
```

여기서 `there`는 말하는 사람과 듣는 사람이 알고 있는 특정 장소를 가리킨다.

## 5-3. find these people a plan

`find + 사람 + 물건` 구조다.

```
find + 간접목적어 + 직접목적어
find + these people + a plan
```

의미는:

```
이 사람들에게 계획 하나를 찾아주다
```

비슷한 구조:

```
Find me a chair.
나한테 의자 하나 찾아줘.
```

## 5-4. that’ll work for them

`that’ll`은 `that will`의 축약형이다.

```
that’ll work for them
= that will work for them
= 그들에게 효과가 있을 / 그들에게 맞을
```

여기서 `that`은 앞의 `a plan`을 가리키는 관계대명사다.

```
a plan that will work for them
그들에게 효과가 있을 계획
```

# 6. 단어/표현 정리

| 단어/표현 | 뜻 | 설명 |
| --- | --- | --- |
| Why don’t you ~? | ~하는 게 어때? / ~해봐 | 제안 또는 부드러운 지시 |
| go in there | 저 안에 들어가다 | 특정 장소 안으로 이동 |
| Dwight | 드와이트 | 상대방 이름을 부르는 호격 |
| find | 찾다, 찾아주다 | 여기서는 해결책을 마련해주다에 가까움 |
| these people | 이 사람들 | 앞에 있는 특정 사람들 |
| plan | 계획 | 해결책, 방안 |
| that’ll | that will | 관계절 안의 축약 표현 |
| work for them | 그들에게 효과가 있다 / 맞다 | 상황이나 사람에게 적합하다는 뜻 |
| okay? | 알겠지? / 괜찮지? | 확인, 동의 요청 |

# 7. 자연스러운 번역

**“드와이트, 저 안에 들어가서 이 사람들한테 맞는 계획 하나 찾아줘, 알겠지?”**

또는 더 자연스럽게:

**“드와이트, 가서 이 사람들한테 통할 만한 방안 좀 찾아봐, 알겠어?”**
