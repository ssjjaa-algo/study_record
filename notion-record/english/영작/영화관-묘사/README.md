## 2. 수정한 글

The movie theater I usually go to is CGV, which is owned by CJ. CGV is really big. There are lots of things we can enjoy there, such as an arcade, McDonald’s, and even karaoke rooms. There are screening rooms and popcorn stands on each floor, so we don’t have to wait in the main lobby. We can just go straight to the floor where our movie is playing.

## 3. 수정 포인트

- **i → I**: 영어에서 “나”는 항상 대문자로 씁니다.
- **The movie theater I always go → The movie theater I usually go to**: `go to`처럼 전치사 `to`가 필요하고, `usually`가 더 자연스럽습니다.
- **CGV is a really big → CGV is really big**: `big`은 형용사라서 앞에 `a`를 쓰지 않습니다.
- **There are lots of things we can enjoy → There are lots of things we can enjoy there**: 장소를 가리키는 `there`를 넣으면 더 자연스럽습니다.
- **play-room → an arcade**: 영화관 안의 게임 공간은 보통 `arcade`라고 합니다.
- **mcdonald → McDonald’s**: 고유명사는 대문자로 쓰고, 브랜드명은 `McDonald’s`가 자연스럽습니다.
- **singer room → karaoke rooms**: 노래방은 영어로 `karaoke room`이라고 표현합니다.
- **층마다 영화관이 있고 → There are screening rooms on each floor**: 영화관 안의 상영관은 `screening rooms` 또는 `theaters`라고 합니다.
- **팝콘 파는 곳 → popcorn stands**: 팝콘을 파는 곳은 `popcorn stands` 또는 `concession stands`라고 합니다.
- **wait a main stage → wait in the main lobby**: 영화관의 대기 공간은 `main lobby`가 자연스럽습니다.
- **just pick it where our movie is start → go straight to the floor where our movie is playing**: “영화가 시작하는 곳으로 가다”는 `where our movie is playing`이 자연스럽습니다.
