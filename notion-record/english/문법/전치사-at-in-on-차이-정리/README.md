# 전치사 at / in / on 차이 정리

# 1. 핵심 이미지

| 전치사 | 핵심 느낌 | 한 줄 설명 |
| --- | --- | --- |
| at | 점, 지점 | 정확한 한 지점에 콕 찍는 느낌 |
| on | 표면, 접촉, 선 위 | 어떤 표면이나 기준선 위에 붙어 있는 느낌 |
| in | 안, 공간, 범위 | 어떤 공간이나 범위 안에 들어가 있는 느낌 |

가장 기본 이미지는 다음과 같다.

```
at = point
on = surface / line
in = space / area / container
```

# 2. 장소에서의 차이

## 2-1. at: 특정 지점

`at`은 장소를 넓은 공간으로 보는 것이 아니라 **하나의 지점**으로 본다.

```
at the bus stop
버스 정류장에서
```

```
at the door
문 앞에서 / 문이라는 지점에
```

```
at school
학교에
```

`at school`은 학교 건물 안쪽 공간을 강조하기보다, **학교라는 활동 장소/지점에 있다**는 느낌이다.

## 2-2. on: 표면 위, 길/선 위

`on`은 어떤 표면에 붙어 있거나, 길/선/층처럼 이어진 기준 위에 있을 때 쓴다.

```
on the table
테이블 위에
```

```
on the wall
벽에
```

```
on the street
거리 위에 / 길가에
```

```
on the second floor
2층에
```

## 2-3. in: 공간 안, 영역 안

`in`은 어떤 공간이나 영역 안에 들어가 있다는 느낌이다.

```
in the room
방 안에
```

```
in the box
상자 안에
```

```
in Seoul
서울 안에 / 서울이라는 지역 안에
```

```
in the office
사무실 안에
```

# 3. 시간에서의 차이

시간에서도 기본 이미지는 같다.

| 전치사 | 시간 느낌 | 예시 |
| --- | --- | --- |
| at | 정확한 시점 | at 5 p.m. |
| on | 특정 날짜/요일 | on Monday |
| in | 넓은 기간 안 | in June |

## 3-1. at: 정확한 시각 / 특정 시점

```
at 7 o’clock
7시에
```

```
at noon
정오에
```

```
at night
밤에
```

```
at the end of the day
하루의 끝이라는 지점에서 / 결국에는
```

`at the end of the day`에서 `at`을 쓰는 이유는 **하루라는 시간 범위의 끝 지점**을 콕 집기 때문이다.

## 3-2. on: 날짜 / 요일 / 특정한 날

```
on Monday
월요일에
```

```
on June 5
6월 5일에
```

```
on my birthday
내 생일에
```

```
on the weekend
주말에
```

미국 영어에서는 보통 `on the weekend`, 영국 영어에서는 `at the weekend`도 자주 쓴다.

## 3-3. in: 월 / 계절 / 연도 / 긴 기간

```
in June
6월에
```

```
in summer
여름에
```

```
in 2026
2026년에
```

```
in the morning
아침에
```

`in the morning`은 아침을 하나의 넓은 시간 구간으로 보기 때문에 `in`을 쓴다.

# 4. 같은 단어와 함께 쓰일 때 의미 차이

## 4-1. school

| 표현 | 의미 | 느낌 |
| --- | --- | --- |
| at school | 학교에 있다 | 학교라는 장소/활동 지점 |
| in school | 재학 중이다 / 학교 안에 있다 | 학교 시스템 안에 있음 또는 건물 안 |
| on school grounds | 학교 부지 위에 | 표면/부지 위 |

예문:

```
He is at school.
그는 학교에 있어.
```

```
He is still in school.
그는 아직 재학 중이야.
```

## 4-2. office

| 표현 | 의미 | 느낌 |
| --- | --- | --- |
| at the office | 회사/사무실에 있다 | 근무 장소라는 지점 |
| in the office | 사무실 안에 있다 | 사무실 내부 공간 |
| on the office floor | 사무실 층/바닥 위에 | 표면/층 위 |

예문:

```
I’m at the office.
나 회사야 / 사무실에 있어.
```

```
I’m in the office.
나 사무실 안에 있어.
```

## 4-3. end

| 표현 | 의미 | 느낌 |
| --- | --- | --- |
| at the end | 끝 지점에서 | 위치/시점 |
| in the end | 결국에는 | 최종 결과 |
| on the end | 끝부분 표면에 붙어 | 물리적 위치 |

예문:

```
At the end of the movie, everyone cried.
영화 끝부분에서 모두가 울었다.
```

```
In the end, we agreed.
결국 우리는 동의했다.
```

# 5. 감각적으로 구분하는 법

## 5-1. at은 점

정확히 한 지점을 찍는 느낌이면 `at`.

```
at the station
역에서
```

역이라는 넓은 내부보다, 만나는 장소로서의 지점을 말한다.

## 5-2. on은 면

어딘가에 붙어 있거나 표면 위에 있으면 `on`.

```
on the screen
화면에
```

```
on the page
페이지에
```

## 5-3. in은 공간

안에 들어가 있거나, 범위 안에 포함되면 `in`.

```
in the picture
사진 속에
```

```
in the city
도시 안에
```

# 6. 자주 쓰는 표현 정리

| at | on | in |
| --- | --- | --- |
| at home | on the table | in the room |
| at work | on the wall | in the car |
| at school | on the street | in Seoul |
| at the door | on Monday | in June |
| at 7 p.m. | on my birthday | in 2026 |
| at the end | on the page | in the morning |
| at this point | on the phone | in the afternoon |

# 7. 헷갈리기 쉬운 표현

## 7-1. on the phone

```
I’m on the phone.
나 통화 중이야.
```

전화기 안에 들어간 것이 아니라, 통신선/연결 위에 있다는 느낌에서 `on`을 쓴다.

## 7-2. in the car vs on the bus

```
in the car
차 안에
```

자동차는 작은 개인 공간처럼 느껴지므로 `in`을 쓴다.

```
on the bus
버스에
```

버스, 기차, 비행기처럼 걸어서 타고 내릴 수 있는 큰 교통수단은 보통 `on`을 쓴다.

## 7-3. at home vs in the house

```
at home
집에 있다
```

집이라는 생활 장소에 있다는 느낌이다.

```
in the house
집 안에 있다
```

물리적으로 집 내부에 있다는 느낌이다.

# 8. 빠른 판단법

어떤 전치사를 쓸지 헷갈릴 때는 이렇게 생각하면 된다.

```
정확한 지점인가? → at
표면이나 선 위인가? → on
공간이나 범위 안인가? → in
```

예시:

```
at the bus stop
버스 정류장이라는 지점
```

```
on the bus
버스라는 이동 수단 위/탑승 상태
```

```
in the bus
버스 내부 공간 안
```

둘 다 가능할 때도 있지만, 강조점이 달라진다.

# 9. 핵심 정리

| 전치사 | 핵심 | 장소 | 시간 |
| --- | --- | --- | --- |
| at | 점 | at the station | at 5 p.m. |
| on | 면/선 | on the table | on Monday |
| in | 공간/범위 | in the room | in June |

가장 중요한 것은 한국어의 “~에” 하나로 외우는 것이 아니라, 영어식으로 **점 / 면 / 공간**의 이미지로 구분하는 것이다.
