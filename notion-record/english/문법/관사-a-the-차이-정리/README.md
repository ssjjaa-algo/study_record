# 관사 a / the 차이 정리

# 1. 핵심 차이

| 관사 | 핵심 느낌 | 의미 |
| --- | --- | --- |
| a / an | 여러 개 중 하나 | 처음 말하는 것, 특정하지 않은 것 |
| the | 바로 그 것 | 서로 알고 있는 것, 특정된 것 |

가장 중요한 차이는 이것이다.

```
a = one of many
여러 개 중 하나
```

```
the = specific one
딱 그 특정한 것
```

# 2. a / an의 기본 느낌

`a`는 **여러 개 중 하나**를 말할 때 쓴다.

```
I saw a dog.
나는 개 한 마리를 봤어.
```

여기서 `a dog`는 어떤 개인지 아직 모른다. 그냥 많은 개들 중 한 마리다.

```
She bought a book.
그녀는 책 한 권을 샀어.
```

책이 어떤 책인지 아직 중요하지 않거나, 듣는 사람이 모르는 책이다.

# 3. the의 기본 느낌

`the`는 **말하는 사람과 듣는 사람이 어느 것인지 알 수 있는 대상**에 쓴다.

```
I saw a dog. The dog was cute.
나는 개 한 마리를 봤어. 그 개는 귀여웠어.
```

처음에는 `a dog`라고 말한다. 아직 듣는 사람은 어떤 개인지 모른다.

하지만 두 번째부터는 이미 언급된 개이므로 `the dog`라고 한다.

```
Close the door.
문 닫아.
```

여기서 `the door`는 아무 문이 아니라, 지금 상황에서 서로 알고 있는 바로 그 문이다.

# 4. a와 the 비교

| 표현 | 의미 | 느낌 |
| --- | --- | --- |
| a dog | 개 한 마리 | 어떤 개인지 특정 안 됨 |
| the dog | 그 개 | 이미 알고 있거나 특정됨 |
| a book | 책 한 권 | 여러 책 중 하나 |
| the book | 그 책 | 말하는 대상이 정해짐 |
| a car | 차 한 대 | 어떤 차든 하나 |
| the car | 그 차 | 특정한 차 |

# 5. 처음 말하면 a, 다시 말하면 the

가장 기본적인 흐름은 다음과 같다.

```
I saw a man.
The man was wearing a black coat.
```

해석:

```
나는 한 남자를 봤어.
그 남자는 검은 코트를 입고 있었어.
```

처음 등장:

```
a man
```

다시 언급:

```
the man
```

# 6. 서로 이미 알고 있으면 처음부터 the

처음 말해도 서로 무엇인지 알 수 있으면 `the`를 쓴다.

```
Where is the bathroom?
화장실 어디 있어요?
```

건물 안에서 말하면, 일반적으로 사용할 수 있는 특정 화장실을 묻는 것이다.

```
The sun is bright today.
오늘 태양이 밝다.
```

태양은 하나뿐이므로 `the sun`이라고 한다.

```
I went to the office.
나 사무실에 갔어.
```

여기서 `the office`는 말하는 사람과 듣는 사람이 아는 특정 사무실이다.

# 7. a는 정체를 모를 때, the는 정체를 알 때

```
A man is waiting outside.
어떤 남자가 밖에서 기다리고 있어.
```

누군지 모르는 남자다.

```
The man is waiting outside.
그 남자가 밖에서 기다리고 있어.
```

이미 말했거나, 서로 알고 있는 남자다.

# 8. 직업에는 보통 a/an

직업은 여러 사람 중 하나의 역할이므로 보통 `a/an`을 쓴다.

```
He is a doctor.
그는 의사야.
```

```
She is an engineer.
그녀는 엔지니어야.
```

`doctor`, `engineer`는 셀 수 있는 단수 명사이므로 관사가 필요하다.

# 9. a와 an의 차이

`a`와 `an`은 의미 차이가 아니라 발음 차이다.

| 관사 | 쓰는 경우 | 예시 |
| --- | --- | --- |
| a | 자음 소리 앞 | a book, a car, a user |
| an | 모음 소리 앞 | an apple, an idea, an hour |

중요한 것은 철자가 아니라 **소리**다.

```
a user
```

`user`는 철자는 u로 시작하지만 발음이 /juː/라서 자음 소리로 시작한다.

```
an hour
```

`hour`는 h가 묵음이고 모음 소리로 시작하므로 `an`을 쓴다.

# 10. the를 쓰는 대표 상황

## 10-1. 이미 언급된 것

```
I bought a phone. The phone was expensive.
나는 휴대폰을 하나 샀어. 그 휴대폰은 비쌌어.
```

## 10-2. 상황상 하나로 특정되는 것

```
Open the window.
창문 열어.
```

지금 말하는 장소에서 어떤 창문인지 알 수 있다.

## 10-3. 세상에 하나뿐인 것

```
the sun
태양
```

```
the moon
달
```

```
the sky
하늘
```

## 10-4. 최상급

```
the best
최고의
```

```
the biggest building
가장 큰 건물
```

최상급은 보통 특정한 하나를 가리키므로 `the`를 쓴다.

# 11. 관사가 없는 경우

모든 명사에 `a`나 `the`가 붙는 것은 아니다.

복수명사나 셀 수 없는 명사를 일반적으로 말할 때는 관사를 쓰지 않을 수 있다.

```
Dogs are friendly.
개들은 친근하다.
```

개 전체를 일반적으로 말한다.

```
Water is important.
물은 중요하다.
```

물을 일반 개념으로 말한다.

하지만 특정하면 `the`를 쓴다.

```
The dogs in this park are friendly.
이 공원에 있는 그 개들은 친근하다.
```

```
The water in this bottle is cold.
이 병 안의 물은 차갑다.
```

# 12. 빠른 판단법

명사를 말할 때 이렇게 판단하면 된다.

```
처음 말하는 하나인가? → a/an
서로 아는 특정 대상인가? → the
일반적인 복수/불가산 개념인가? → 관사 없음
```

예시:

```
I need a pen.
펜 하나 필요해.
```

아무 펜 하나면 된다.

```
I need the pen.
그 펜이 필요해.
```

특정한 펜이 필요하다.

```
I like pens.
나는 펜을 좋아해.
```

펜이라는 물건 일반을 말한다.

# 13. 핵심 정리

| 상황 | 관사 | 예시 |
| --- | --- | --- |
| 여러 개 중 하나 | a/an | a book |
| 이미 언급된 것 | the | the book |
| 서로 알고 있는 특정 대상 | the | the door |
| 직업, 정체 | a/an | a doctor |
| 일반 복수 | 관사 없음 | dogs |
| 일반 불가산 | 관사 없음 | water |
| 특정 복수/불가산 | the | the dogs, the water |

가장 중요한 감각은 이것이다.

```
a/an = 아무거나 하나, 여러 개 중 하나
```

```
the = 바로 그거, 서로 아는 특정 대상
```
