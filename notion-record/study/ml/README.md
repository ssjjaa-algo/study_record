# ML

# 머신 러닝의 정의

- 명시적 프로그래밍 없이 컴퓨터가 학습하는 능력을 갖추게 하는 연구 분야
- Task T에 대한 컴퓨터 프로그래밍 성능을 P로 측정했을 때 경험 E로 인해 성능이 향상됐다면, 이 컴퓨터 프로그램은 작업 T와 성능 P에 대해 경험 E로 학습한 것이다.
    - 성능 P가 어떤 것인지는 직접 정의 **ex) 정확도**

## 데이터마이닝

- 대용량의 데이터를 분석해서 숨겨진 패턴을 발견하는 것

# 훈련 지도 방식

## 지도 학습

- 답을 알고 훈련을 한다 (= 답이 정해져있다)
- 레이블
    - 분류(classification)에서 쓰이는 용어로 원하는 답을 말한다.
    - 학습 데이터에서 모델이 예측해야 하는 **정답**이나 **목표값**을 의미
        
        **ex) A는 스팸메일이 맞다. 아니다.**
        
- 타겟
    - 회귀(regression)에서 쓰이는 용어로 레이블과 동의어로 본다.
        - feature를 통해 target을 예측
        - *참고: regression의 어원 → 프랜시스 골턴
    - **ex) feature로 부터(자동차 회사, 부품, 연식 등) →  target 예측(중고가는 1000만원일 것이다)**

## 비지도 학습

- 군집 알고리즘, 시각화, 이상치 탐지등에 쓰임.
- 레이블 없이 훈련시킨다.
- 모호 1. 군집 알고리즘
    - 군집 알고리즘을 통해 1차 그룹화
    - 계층 군집 알고리즘을 사용해서 그룹을 더 작은 그룹으로 세분화(2차 그룹화)
- 모호 2. 차원 축소(Dimensionality reduction)와 특성 추출(Feature Extraction)의 개념
    - feature extraction, feature selection 차이와 함께 봐야할듯
    - 특성을 하나로 합쳐서 판단 요소를 줄인다.

## 준지도 학습

- 레이블이 있는 것도 없는 데이터가 혼재되어있을 때 사용하는 학습방식
- 대부분은 지도학습과 비지도 학습의 조합으로 이뤄짐
- **ex) 구글 사진 인물 구별**
    - 분류와 군집을 사용

## 자기지도 학습

- 레이블이 하나도 없는 데이터셋에서 레이블이 완전히 부여된 데이터셋을 만든다
    - 레이블이 없는 데이터에서 **데이터 자체로부터 자동으로 학습 신호를 만들어내는** 학습 방법
    - 레이블이 없는 이미지 일부를 마스킹하고 모델이 복원하도록 훈련시킨다
        - **ex) 고양이 사진 얼굴 지워놓고 복구하라고 함. 이때 원본 사진을 레이블로 활용**
- 최종적으로는
1. 레이블 없는 데이터로 이미지 복원 모델 훈련
2. 레이블이 있는 데이터셋에서 모델을 미세튜닝 
- 훈련하는 동안 (생성된) 레이블을 사용한다 → 자기 지도 학습에 가깝다..

### 자기지도 학습과 비지도 학습과의 차이

- **일반 비지도 학습**: 데이터의 구조나 패턴을 발견 (군집화 등)
- **자기지도학습**: 데이터 자체에서 생성한 "가짜 레이블"로 지도 학습처럼 훈련
