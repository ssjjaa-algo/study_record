# 자바(Java Programming Language)

## 개념

- 썬 마이크로시스템즈에서 개발한 **객체지향** 프로그래밍 언어

## 특징

- **운영체제에 독립적**
    - 운영체제나 하드웨어가 아닌 JVM하고만 통신
    - 그러나 **JVM은 운영체제에 종속적**임
    - 썬에서는 서로 다른 버전의 JVM을 제공
- **객체지향 언어**
    - 상속, 캡슐화, 다형성
- **Garbage Collector**
    - 자동으로 메모리 관리
- **네트워크와 분산처리 지원**
- **멀티쓰레드**
- **동적 로딩**
    - 실행 시에 모든 클래스가 로딩되지 않고 필요한 시점에 클래스를 로딩하여 사용

## 특징 2 : JVM

- 자바를 실행하기 위한 가상 machine
    - 가상 기계?
        - 소프트웨어로 구현된 하드웨어를 뜻하는 넓은 의미의 용어
- 자바로 작성된 애플리케이션은 모두 JVM에서만 실행된다
- 일반 애플리케이션의 코드는 OS만 거치고 코드로 전달
    - JAVA는 이 사이에 JVM을 한 번 더 거친다.
    - 실행 시에 해석되기 때문에 속도가 느림.
        - JIT컴파일러 등이 속도를 해결해주고 있다.
