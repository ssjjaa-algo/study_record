# 형변환(캐스팅, casting)

# 개념

- 변수 또는 상수의 타입을 다른 타입으로 변환
    - int와 float를 더한다 → 두 값을 같은 타입으로

# 방법

<aside>
💡 (타입)피연산자
double d = 85.4;
int score = (int) d;

</aside>

- 기본형과 참조형의 형변환 불가

# 정수형 간의 형변환

- int → byte : 값 손실 발생 가능성

# 실수형 간의 형변환

- 작은 타입 → 큰 타입 : 빈 자리를 0으로 채운다.

# 자동 형변환

- 기존의 값을 최대한 보존할 수 있는 타입으로 자동 형변환
- 반대 방향으로의 변환은 반드시 형변환 연산자를 사용
