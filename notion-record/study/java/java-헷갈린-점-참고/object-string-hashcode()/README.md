# Object / String hashCode()

해싱을 구현한 클래스에서는 Object 클래스에 정의된 hashCode를 해시함수로 사용

## Object 클래스의 hashCode() 메서드

- 모든 객체에서 상속되는 기본 구현을 제공
- **`객체의 내부 메모리 주소를 기반`**으로 해시 코드를 생성
- 즉, 각 객체는 **`고유한 메모리 주소`**를 가지고 있으므로,
- **`서로 다른 객체는 서로 다른 해시 코드를 반환`**

## String 클래스의 hashCode() 메서드

- 문자열의 내용을 기반으로 해시 코드를 생성.
- 문자열의 각 문자에 대해 일정한 해시 값을 계산하고, 이들을 결합하여 최종 해시 코드를 생성
- 동일한 내용을 가진 문자열은 항상 동일한 해시 코드를 반환
- 문자열의 내용이 같으면 동일한 해시 코드를 가지도록 보장하기 위해 설계

Object 클래스의 hashCode() 메서드는 **`주로 객체의 고유성 비교`**에 사용

 String 클래스의 hashCode() 메서드는 **`문자열의 동등성 비교`**에 사용
